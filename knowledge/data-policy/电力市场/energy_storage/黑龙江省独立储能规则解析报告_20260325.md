## 一、新型主体基本情况

### 1.1 定义与核心特征

**定义**：独立新型储能电站是指在黑龙江省境内，独立于新能源项目或其他发电设施，以独立法人资格接入电网、具备独立计量和控制调度条件，可自主参与电力市场交易的集中式储能设施。

**政策依据**：
- 《黑龙江省电力市场运营规则》（2025年12月30日，"1+6"规则体系）
- 《黑龙江省电力中长期交易规则》补充规定（东北能源监管局，2025年6月6日）
- 《黑龙江省新型储能规模化建设专项实施方案（2025-2027年）》（黑发改新能源〔2025〕990号，2025年11月21日）

**核心特征**：

| 序号 | 特征 | 说明 | 政策依据 |
|------|------|------|---------|
| 1 | 独立法人资格 | 必须具有独立法人资格，可独立签订交易合同 | 中长期交易规则补充规定 |
| 2 | 独立并网调度 | 与电网企业签订并网调度协议，接入调度自动化系统 | 同上 |
| 3 | 双重身份 | 既可作为买方（充电/用电），也可作为卖方（放电/发电） | 市场运营规则 |
| 4 | 公平输电服务 | 获得公平的输电服务和电网接入服务 | 中长期交易规则补充规定 |
| 5 | 统一调度 | 服从电力调度机构的统一调度 | 同上 |
| 6 | 配建可转独立 | 配建储能具备独立计量/控制/调度条件时可转为独立储能 | 黑发改新能源〔2025〕990号 |

## 三、交易规则

### 3.1 交易品种

| 交易品种 | 参与方式 | 交易周期 | 申报规则 | 政策依据 |
|---------|---------|---------|---------|---------|
| **现货电能量市场** | "报量不报价"或"报量报价" | 日前(D-1)、实时(D日) | 申报充放电出力段 | 市场运营规则 |
| **中长期交易** | 分时段签约 | 年、月 | "分时段交易、分时段计量、分时段结算" | 市场运营规则 |
| **调频辅助服务** | 报量报价 | 日前报价→日内分小时出清 | 调频容量≤额定功率比例g3 | 辅助服务(调频)实施细则 |
| **跨省备用** | 报量报价 | 东北区域市场 | 东北区域规则 | 东北区域辅助服务规则 |
| **容量补偿** | 自动获得 | 月度核定 | 仅限纳入建设清单项目 | 黑发改新能源〔2025〕990号 |

### 3.2 现货市场参与方式

| 维度 | 报量不报价 | 报量报价 |
|------|-----------|---------|
| **适用阶段** | 初期试水、策略简单 | 成熟运营、追求收益最大化 |
| **价格接受** | 按市场出清价格结算 | 自主报价，影响出清结果 |
| **风险** | 低（被动接受价格） | 中（报价不当可能亏损） |
| **推荐** | ⭐ 建议初期采用 | 运营成熟后切换 |

> 📌 **黑龙江特色**：独立储能可选择"报量不报价"参与现货市场，降低了参与门槛和策略复杂度。这是区别于山东（仅报量报价）的重要政策红利。

### 3.3 申报规则

| 申报维度 | 规则 | 说明 |
|---------|------|------|
| 买方身份（充电） | ✅ 可申报 | 独立储能以用电身份充电 |
| 卖方身份（放电） | ✅ 可申报 | 独立储能以发电身份放电 |
| 充放电出力段 | 按规则分段申报 | 参照市场运营规则具体细则 |
| 中长期合约 | 无强制签约比例要求 | 灵活签约 |
| 价格限制 | 现货价格上下限 | 参照市场运营规则限价 |

### 3.4 调频辅助服务

| 参数 | 标准 | 说明 |
|------|------|------|
| 调频类型 | 区域调频（AGC） | 含一次调频 |
| 调频容量申报 | ≤额定功率一定比例g3 | 具体比例以细则为准 |
| 里程出清价格上限 | ≤0.015元/kW | 出清价格限制 |
| 结算方式 | 按里程结算 | 实际调用后获得里程补偿 |
| 未中标被调用 | 按市场边际出清价格结算 | - |

## 四、调度与运营

### 4.1 调度管理

| 调度类型 | 调度机构 | 调度指令 | 响应要求 | 政策依据 |
|---------|---------|---------|---------|---------|
| 日前调度 | 省调 | D-1日下发发电/充电计划 | 按计划执行 | 市场运营规则 |
| 日内/实时调度 | 省调 | 实时调整指令 | 即时响应 | 市场运营规则 |
| 辅助服务调度 | 省调/东北区域调 | AGC/调频指令 | 按辅助服务规则 | 辅助服务实施细则 |

### 4.2 东北区域"两个细则"考核

2024年10月，东北能源监管局正式印发新版"两个细则"，新型储能电站被纳入管理范围：

| 考核维度 | 考核内容 | 说明 |
|---------|---------|------|
| **并网运行管理** | 发电计划曲线偏差、非计划停运、AGC、一次调频、AVC、检修、继电保护 | 与火电机组类似 |
| **调度纪律** | 未执行调度指令、迟报/瞒报信息 | 新版细则新增 |
| **AGC性能** | 调节速率、调节精度、响应时间 | 调频市场核心指标 |
| **一次调频** | 超过理论动作积分电量70%部分可获得补偿 | 补偿标准参照东北区域规则 |
| **计划曲线** | 日前计划与实际运行偏差考核 | 新版细则新增 |

### 4.3 东北区域辅助服务补偿标准（参考）

> ⚠️ 以下为东北区域历史参考标准，2024年10月新版"两个细则"已有调整，具体以最新文件为准。

| 辅助服务品种 | 补偿标准（参考） | 分摊方式 |
|------------|----------------|---------|
| **AGC调频** | 按调整电量补偿（每万千瓦时约1200元） | 发电企业+储能+电力用户共同分摊 |
| **一次调频** | 小频差：超过70%理论积分电量的0.5×R1元/MWh | 按月度结算 |
| **深度调峰** | 160元/MWh（新型储能参考标准） | - |
| **调峰（填谷）** | 参照《东北电力辅助服务市场运营规则》 | 新能源+核电+未达基准火电分摊 |
| **调峰（顶峰）** | 参照《东北电力辅助服务市场运营规则》 | 全体市场化用户分摊 |
| **黑启动** | 新版细则明确储能可参与 | 特殊情况调用 |

### 4.4 运营要求

| 要求 | 说明 |
|------|------|
| **5MW/1h纳入管理** | 电化学储能容量达到5MW/1小时之日起纳入并网运行和辅助服务管理 |

## 五、结算规则

### 5.1 结算流程

```
计量（D+1日）→ 核算（D+2日）→ 审核（D+3日）→ 结算（D+5日）→ 支付（月结）
```

| 步骤 | 负责部门 | 办理时限 | 说明 | 政策依据 |
|------|---------|---------|------|---------|
| 1. 电量计量 | 电网企业/计量装置 | 实时/5分钟级 | 分时计量充放电电量 | 计量管理实施细则 |
| 2. 数据核算 | 黑龙江电力交易中心 | D+2日 | 核算各品种电量和费用 | 电费结算实施细则 |
| 3. 审核确认 | 电力交易中心+电网企业 | D+3日 | 审核无误后确认结算单 | 同上 |
| 4. 出具结算单 | 黑龙江电力交易中心 | 月度 | "日清月结" | 中长期交易规则 |
| 5. 费用支付 | 电网企业 | 按月 | 电费结算 | 同上 |

### 5.2 结算价格

| 结算品种 | 结算价格 | 说明 |
|---------|---------|------|
| **现货电能量** | **实时节点电价** | 实际充放电电量按实时节点电价结算 |
| **中长期合约** | **合约约定价格** | 中长期合约电量按合约价格结算 |
| **偏差电量** | **现货市场价格** | 中长期合约偏差电量按现货价格差价结算 |
| **调频里程** | **出清价格** | 按市场边际出清价格结算 |
| **容量补偿** | **待公布** | 分类研究标准，按年公布 |

### 5.3 ⭐ 保底补偿机制（黑龙江独有）

这是黑龙江独立储能政策中**最具吸引力的条款**：

> **独立储能因现货价差偏小，导致每万千瓦容量日均电能量收益不足0.765万元的，予以补偿。**

即：**保底补偿标准 = 0.765元/kW·日**

| 维度 | 说明 |
|------|------|
| **触发条件** | 日均电能量收益 < 0.765元/kW |
| **补偿方式** | 差额补足至0.765元/kW |
| **测算** | 50MW电站年保底 = 0.765 × 50,000 × 365 = **1,396万元/年** |
| **电网原因** | 因电网原因导致运行曲线不一致，偏差盈利自留，损失按日前应得收益补偿 |
| **政策依据** | 《黑龙江省电力中长期交易规则》补充规定（2025年6月） |

### 5.4 偏差结算规则

| 维度 | 规则 |
|------|------|
| 结算方式 | "照付不议、偏差结算、日清月结" |
| 中长期偏差 | 偏差电量按市场化价格进行差价结算 |
| 24时段解耦 | 发、用两侧均按照24时段解耦结算 |
| 现货运行期间 | 按现货运行阶段规则执行偏差结算 |

## 七、实操要点

### 7.1 实操案例

**案例 1：黑龙江2026年独立储能建设清单项目**

- **背景**：2026年2月14日，黑龙江省发改委公布年度独立储能建设清单
- **规模**：16个项目，总规模1675MW/6117.2MWh
- **项目构成**：14个电化学储能（含1个钠离子混合）+ 2个电化学+飞轮混合
- **投资方**：远景能源、中能建投、运达股份、海博思创、大连融科储能集团等
- **关键要求**：并网前至并网后一年内不得转让股份或更改投资主体
- **启示**：
  1. 黑龙江独立储能市场正在快速启动，2026年1月备案规模已达10.84GW/38.26GWh
  2. 股权锁定要求（1年）表明政府希望防止投机性转让
  3. 纳入建设清单是获取容量补偿的前提条件

**案例 2：配建储能转独立储能**

- **背景**：黑龙江鼓励新能源配建储能转为独立储能，不得将配置储能作为新建新能源项目的前置条件
- **操作路径**：
  1. 确认配建储能具备独立计量、控制、调度条件
  2. 获取所属新能源项目法人同意
  3. 成立独立法人实体（或使用已有法人）
  4. 与电网签订独立并网调度协议
  5. 在黑龙江电力交易中心注册为独立储能主体
  6. 转入商业运营次月起参与市场交易
- **启示**：存量配建储能盘活是一条低成本的入市路径

### 7.3 常见问题（FAQ）

**Q1：黑龙江独立储能的准入门槛是多少？**
A：额定功率≥5MW，持续充放电时间≥1小时（充放电均），需签订并网调度协议并投入商业运营。

**Q2：保底补偿0.765元/kW·日是如何计算的？**
A：如果独立储能因现货价差偏小，导致每万千瓦容量日均电能量收益不足0.765万元（即7650元），差额部分由政策补偿。以50MW为例，日均保底为50×7650=38.25万元/日。

**Q3：如何获取容量补偿？**
A：必须纳入2025-2027年独立新型储能项目建设清单。2026年清单已公布（16个项目/1675MW），建议争取2027年批次。先并先得。

**Q4：配建储能如何转为独立储能？**
A：需满足四个条件：独立计量、独立控制、独立调度、独立法人资格，并获得所属主体法人同意。

**Q5：黑龙江独立储能可以选择不报价参与现货吗？**
A：可以。独立储能可选择"报量不报价"方式参与现货市场，降低策略复杂度。

**Q6：寒地储能需要什么特殊技术？**
A：电池系统需满足低温充放电性能要求，建议选用磷酸铁锂+液冷系统，并配置保温/加热装置。钠离子电池在低温下性能更优，是黑龙江的优选技术路线之一。
