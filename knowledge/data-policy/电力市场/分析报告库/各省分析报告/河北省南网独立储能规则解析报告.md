## 第二章 市场准入与注册

### 2.1 准入条件

| 序号 | 条件 | 具体要求 |
|------|------|---------|
| 1 | 法人资格 | 具有独立法人资格，财务独立核算 |
| 2 | 规划纳入 | 须列入省级规划，依法取得项目核准或备案文件 |
| 3 | 并网验收 | 通过并网验收或安全性评价 |
| 4 | 并网调度协议 | 与电网企业签订，接入调度自动化系统 |
| 5 | AGC装置 | 须装设AGC装置，性能指标满足调度要求，可独立控制 |
| 6 | 独立计量 | 具备电力、电量数据分时计量与传输条件，数据准确满足结算要求 |
| 7 | **最小功率** | 最大充放电功率≥**10MW** |
| 8 | **最小容量** | 调节容量≥**20MWh**，持续充放电时间≥**2小时** |

### 2.2 市场注册流程

```
省级规划纳入 → 项目核准/备案 → 并网验收 → 签订并网调度协议
→ 在电网企业营销系统完成报装立户 → 电力交易平台注册（公示5个工作日）
→ 签订入市协议/承诺书 → 正式参与市场
```

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 并网调度协议 | 明确调度关系和技术要求 |
| 2 | 报装立户 | 在电网企业营销系统完成 |
| 3 | 平台注册 | 河北电力交易平台注册，公示5个工作日 |
| 4 | 身份认证 | 办理数字安全证书或同等安全等级身份认证 |
| 5 | 入市协议 | 公示无异议后签订 |
| 6 | 退出提前45日 | 参与交易后退市需提前45个工作日申请 |

## 第三章 新能源配储政策

### 3.1 配储政策分界线（核心政策节点）

> ⭐ 2025年2月9日是河北南网新能源配储政策的**最重要分界线**。

| 项目批复时间 | 配储要求 | 政策性质 |
|------------|---------|---------|
| **2025年2月9日前**（含存量项目） | **全生命周期强制配建或租赁储能** | 强制 |
| **2025年2月9日后**（新增项目） | **不再要求强制配置储能**，鼓励租赁独立储能 | 自愿 |

### 3.2 存量项目配储要求（2025年2月9日前批复）

| 维度 | 规则 |
|------|------|
| 配储方式 | 全生命周期配建（自建）或租赁独立储能容量 |
| 容量要求 | 须同时满足**充放电功率**和**时长**要求 |
| 租赁约束 | 租赁容量须同时满足充放电功率和时长要求 |
| 最低时长 | 参照独立储能标准，持续充放电≥2小时 |
| 未配储后果 | 未配建储能的，须按要求与独立储能企业开展容量租赁交易 |

### 3.3 新能源项目配储规格参考

河北南网对不同类型新能源项目配储要求参考：

| 项目类型 | 配储比例参考 | 配储时长 |
|---------|------------|---------|
| 集中式风电 | 按省级年度方案要求 | 原则上≥2小时 |
| 集中式光伏 | 按省级年度方案要求 | 原则上≥2小时 |
| 分布式光伏/分散式风电 | 按当地配储规定 | — |

> 具体配储比例和时长以每批次项目批复文件为准。

### 3.4 136号文承接方案（2025年9月）

河北省贯彻国家《关于深化新能源上网电价市场化改革的通知》（发改价格〔2025〕136号）：

| 维度 | 内容 |
|------|------|
| 存量项目 | 全生命周期配建或租赁储能，机制电量比例按类别设定上限（集中式风电70%、光伏40%等） |
| 增量项目 | 不再要求强制配置储能，鼓励租赁独立储能容量提升调节性能 |
| 机制电价 | 按燃煤基准价0.3644元/kWh执行 |
| 约束 | 不得向新能源不合理分摊费用 |

## 第四章 容量租赁市场

### 4.1 容量租赁机制定位

| 维度 | 规则 |
|------|------|
| 制度来源 | 2026年《电力市场运行规则》正式将容量租赁纳入电力中长期市场管理 |
| 交易标的 | 独立储能提供的能够可靠支撑最大负荷的**出力能力**（容量） |
| 出租方 | 独立储能企业 |
| 租赁方 | 有配储要求的新能源发电企业 |
| 租赁期限 | 最短一自然年；首次并网新能源项目租赁期至少到当年年底；全年容量和价格原则上保持一致 |
| 多年期 | 鼓励开展多年期容量租赁交易 |

### 4.2 容量租赁价格机制

| 年份 | 价格区间 | 依据 |
|------|---------|------|
| **2025年** | 下限200元/kW·年，上限400元/kW·年 | 2025年中长期交易方案 |
| **2026年** | 以2025年政策为基础，具体以当年方案为准 | 待当年方案明确 |

> 💡 市场实际成交价：2026年2月河北邢台某150MW风电项目，30MW/60MWh租赁成交价约**432万元/约11个月**，折合约**157元/kW·年**。

### 4.3 租赁容量计算规则

| 维度 | 规则 |
|------|------|
| 可出租容量 | = 装机容量（暂不考虑容量衰减） |
| 出租上限 | 累计出租容量不得超过装机容量 |
| 扣减机制 | 每次达成交易后，相应扣减可出租容量，直至装机容量全部扣除 |
| 新能源容量折算 | 租赁容量须同时满足充放电功率和时长；功率和时长不满足时按比例折算容量差额 |

### 4.4 未足额租赁的处罚

| 情形 | 规则 |
|------|------|
| 新能源企业未足额租赁 | 容量差额部分按当月全网容量租赁**均价×1.2倍**强制支付 |
| 独立储能可用容量不足 | 由调度机构出具备忘，按日累计容量和当月均价×1.2倍扣除其容量租赁费用 |
| 差额费用去向 | 按市场化结算电量比例向**燃煤发电企业**分配 |

## 第五章 容量电价机制

### 5.1 容量电价总体框架

| 维度 | 规则 |
|------|------|
| 政策依据 | 冀发改能价〔2025〕366号（2025年1月1日起实施） |
| 制度定位 | 临时性支持政策，引导独立储能投资 |
| 适用范围 | 河北南网+冀北电网各自管理 |
| 适用门槛 | 容量≥**10万kW**，满功率持续放电≥**2小时** |
| 容量规模 | 河北南网**770万kW**（2025-2026年期间），冀北830万kW |
| 分配方式 | **竞争方式**，全容量并网时间先后排序，**先建先得** |

### 5.2 容量电价标准

| 时间段 | 年度容量电价 | 月度标准 | 说明 |
|--------|-----------|---------|------|
| **2025年1月-2026年12月** | **100元/kW·年**（含税） | 8.3333元/kW | 完整执行 |
| 2024年已执行退坡的 | 不足100元/kW的追补至100元/kW | — | 补差机制 |
| 激励时限（按期并网） | 24个月 | 自商业运营次月起 | 含2024年已执行项目 |
| 激励时限（逾期并网） | 24个月扣减逾期月份 | — | 不足一个月按一个月算 |

### 5.3 容量认定与考核

**月度可用容量计算公式**：

```
月度平均可用容量 = ∑[0.5 × (日可用充电容量 × 可持续充电时长 + 日可用放电容量 × 可持续放电时长) / 4小时] / 当月总天数
```

> 统一以4小时充放电时长为基准折算，这是储能可用容量的核心考核口径。

**考核机制（按月考核）**：

| 违规次数 | 扣减比例 |
|---------|---------|
| 月内发生**2次** | 扣减当月容量电费的**10%** |
| 月内发生**3次** | 扣减当月容量电费的**50%** |
| 月内发生**4次及以上** | 扣减当月容量电费的**100%** |

**分摊方式**：独立储能容量电费纳入**系统运行费**，由全体工商业用户按月分摊。

### 5.4 充放电价格政策

| 阶段 | 充电电价 | 放电电价 |
|------|---------|---------|
| 直接参与市场前（代理购电阶段） | 按工商业代理购电价格执行 | 按当月代理购电市场化采购平均价格（平段价格）结算 |
| 进入现货市场前（中长期阶段） | 作为用电主体参与中长期交易 | 作为发电主体参与中长期交易 |
| 进入现货市场后 | 按现货市场价格结算 | 按现货市场价格结算 |

> **充电免输配电费**：独立储能向电网送电时，其充电电量**不承担输配电价、系统运行费用和政府性基金及附加**（不执行功率因数考核，按规定承担上网环节线损费用）。

## 第六章 交易机制

### 6.1 交易品种体系

| 交易类型 | 具体品种 | 参与方式 |
|---------|---------|---------|
| 电能量交易 | 电力中长期交易（双边协商、集中竞价、滚动撮合、挂牌） | 自愿参与，申报曲线和价格 |
| 容量租赁交易 | 年度/月度/多年期容量租赁 | 独立储能作为售方，新能源作为购方 |
| 现货市场 | 日前+实时现货电能量市场 | 报96点充放电曲线+SOC初始值，不申报价格 |
| 辅助服务 | 调频辅助服务市场（市场初期主要品种） | 满足≥10MW充电功率可参与 |

### 6.2 现货市场参与规则（河北南网2026年）

| 维度 | 规则 |
|------|------|
| 申报内容 | 须申报次日**SOC初始值** + **96点充/放电（发/用电）曲线** |
| 价格申报 | **不申报价格**（作为价格接受者） |
| 未申报后果 | 实时现货市场全天按**零出清** |
| 参与方式 | 可选择仅参与日前市场（自愿）；也可参与日前+实时 |
| 可用容量申报 | 须日前完成运行日可用充/放电容量及时长申报，与实时市场申报曲线保持一致 |
| 调度考核 | 接受实际可用容量的抽查和考核 |
| 未参与调频时 | 实时现货市场沿用日前可靠性机组组合模式下的充放电曲线 |
| 同时参与调频时 | 按调频市场实时调用和AGC指令执行 |

### 6.3 调频辅助服务市场（2026年新开）

| 维度 | 规则 |
|------|------|
| 市场定位 | 市场初期主要开展调频辅助服务 |
| 准入门槛 | 参与调频的独立储能**充电功率≥10MW** |
| 速率上限 | 独立储能等新型主体中标调频总速率占比不超过调频总速率需求的**k倍（k=0.3）** |
| 里程补偿 | 按调频里程和出清价格计算 |
| 出清价格 | 统一边际出清 |
| 上限 | 调频里程申报/出清价格上限**15元/MW** |

### 6.4 2026年价格限值参数

| 参数类型 | 数值 |
|---------|------|
| 中长期曲线交易（燃煤） | 基准价364.4元/MWh ±20% |
| 中长期其他主体申报 | **0-1200元/MWh** |
| 分时交易申报 | **0-1200元/MWh** |
| 现货申报价格 | **0-1200元/MWh** |
| 现货出清价格 | **0-1200元/MWh** |
| 调频里程出清上限 | **0-15元/MW** |
| 容量租赁下限（2025年） | **200元/kW·年** |
| 容量租赁上限（2025年） | **400元/kW·年** |

## 第七章 结算规则

### 7.1 结算模式

| 维度 | 规则 |
|------|------|
| 结算周期 | 按自然月结算 |
| 偏差结算 | "照付不议、偏差结算"原则 |
| 合约结算 | 按约定电量、电价结算 |
| 偏差价格 | 实际执行与合约偏差按偏差价格结算 |
| 结算依据 | 电力交易机构按自然月出具结算依据 |

### 7.2 容量租赁费结算

| 维度 | 规则 |
|------|------|
| 结算方式 | 在独立储能主体、新能源企业上网电费中**增加或扣减**相应电费 |
| 结算周期 | 按月与发电企业电能量电费**一并结算** |
| 未足额租赁 | 差额部分按当月均价×1.2倍强制支付，由新能源企业承担 |
| 储能可用不足扣费 | 按日累计容量和当月均价×1.2倍扣除，差额向燃煤企业分配 |

### 7.3 电能量结算

| 阶段 | 充电电费 | 放电电费 |
|------|---------|---------|
| 直接参与市场前 | 按代理购电价格 | 按代理购电市场化采购均价（平段） |
| 中长期阶段 | 作为用电主体参与 | 作为发电主体参与中长期 |
| 现货阶段 | 按现货市场价格 | 按现货市场价格 |

### 7.4 独立储能可获得的全部收入项

| 序号 | 收入项 | 说明 |
|------|-------|------|
| 1 | 容量租赁费 | 出租给新能源企业的容量费用 |
| 2 | 容量电费 | 100元
| | 电费 | 元/kW·年，考核后发放 |
| 3 | 放电电能量电费 | 现货或中长期放电收益 |
| 4 | 充电电能量电费 | 现货或中长期充电成本 |
| 5 | 调频里程补偿 | 参与调频市场时获得 |
| 6 | 成本补偿 | 启动成本、空载成本（收入无法覆盖时触发） |

## 第八章 投资收益测算

### 8.1 典型项目假设（100MW/400MWh，4小时）

| 参数 | 数值 |
|------|------|
| 装机功率 | 100MW |
| 装机容量 | 400MWh（4小时） |
| 出租容量假设 | 假设60MW出租（其余自用） |
| 年租金（容量租赁） | 60MW × 157元/kW·年 = 942万元 |
