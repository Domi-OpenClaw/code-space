### 1.2 定义与核心特征

**定义**：浙江省独立新型储能电站是指具备独立法人资格、直接接入电网运行、以独立市场主体身份参与电力市场的电化学储能设施。不依附于发电企业或用户侧项目，单独履行注册入市手续，独立参与电能量市场和辅助服务市场。

**政策依据**：
- 《浙江电力现货市场规则》（2024年4月正式印发）
- 《浙江电力现货电能量市场交易实施细则（3.0版）》（2025年11月，2026年1月实施）
- 《浙江电力调频辅助服务市场交易实施细则（3.0版）》（2025年11月28日，2026年1月实施）
- 《浙江电力市场结算实施细则（3.1版）》（2026年2月，2026年5月实施）
- 《浙江电力市场管理实施细则（3.1版）》（2026年2月，2026年5月实施）
- 《2026年浙江电力现货市场运行方案》（2025年11月27日，2026年全年执行）

**核心特征**：

| 序号 | 特征 | 说明 | 政策依据 |

### 1.4 重要澄清：按小时选择机制（非"永久二选一"）

我方首版报告中表述为"储能须在调频市场与电能量市场中选择其一参与"，这一表述存在**重大误解**。正确规则如下：

> **正确表述**：已完成注册的储能电站在每日D-1日申报阶段，**按小时**选择当日各小时是参与电能量市场还是调频市场。入选调频的时段不参与电能量市场；其余时段按申报的自计划曲线参与电能量市场。**储能调频容量之和上限为200MW，超出上限的储能继续参与电能量市场。**

这一机制的实质是：**每日按小时动态调配储能资源**，而非一次性在两种市场间做永久选择。

## 二、市场准入规则

### 2.1 准入条件

| 条件类型 | 具体要求 | 政策依据 |
|---------|---------|---------|
| **额定功率** | 参照浙江新型储能项目管理办法 | 《浙江省新型储能项目管理办法（暂行）》征求意见稿 |
| **持续充放电时间** | 额定功率低于5MW的用户侧储能不纳入规模管控 | 同上 |
| **并网测试** | 完成并网测试且检测结果合格，认定具备充放电能力 | 现货实施细则3.0版 |
| **涉网试验** | 完成涉网试验且试验结果合格，认定具备自动调节和辅助服务能力 | 现货实施细则3.0版 |
| **调度控制** | 可接受调度部门运行控制 | 现货实施细则3.0版 |
| **容量要求** | 扣除租赁部分容量后方可参与市场（若涉及租赁） | 现货实施细则3.0版 |
| **注册手续** | 在浙江电力交易中心完成注册 | 市场管理实施细则3.1版 |
| **履约保证** | 按月满足履约保证初始额度要求（新增，3.1版） | 市场管理3.1版 |
| **信用评级** | 按时缴纳欠费，否则扣分直至降级（新增，3.1版） | 市场管理3.1版 |

### 2.2 电网侧储能特别说明

| 维度 | 规则 | 说明 |
|------|------|------|
| 参与方式 | **自愿参与**现货市场，非强制 | 2026年运行方案明确 |
| 注册要求 | 满足准入条件，在交易中心完成注册 | 市场管理3.1版 |
| 容量管理 | 扣除租赁部分容量 | 现货实施细则 |
| 配建储能 | 可选择转为独立储能项目，具有法人资格时 | 市场管理3.1版 |

> 💡 **"自愿参与"的含义**：2026年运行方案明确"电网侧储能：自愿参与现货市场"。这意味着独立储能可以根据自身经营策略和市场行情，自主选择是否参与现货市场，以及参与的程度。但若选择参与，则须遵守市场规则。

### 2.3 注册流程

```
并网测试（认定充放电能力）→ 涉网试验（认定辅助服务能力）→ 浙江电力交易中心注册 → 签订入市协议 → 按月维持履约保证额度（3.1版新增）→ 参与市场
```

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 并网测试 | 认定具备充放电能力，可接受调度控制 |
| 2 | 涉网试验 | 认定具备自动调节和提供辅助服务能力 |
| 3 | 交易中心注册 | 提交相关材料，获取市场主体资格 |
| 4 | 签订入市协议 | 明确权利义务 |
| 5 | 履约保证（3.1版新增） | 按月满足履约保证初始额度要求 |
| 6 | 正式参与市场 | 按小时选择电能量或调频市场 |

## 三、交易规则

### 3.1 交易品种

| 交易品种 | 参与方式 | 申报规则 | 政策依据 |
|---------|---------|---------|---------|
| **电能量市场（现货）** | 自计划方式申报曲线 | 按小时申报D日出清曲线 | 现货实施细则3.0版+2026年运行方案 |
| **调频辅助服务市场** | 报量报价，里程出清 | D-1日申报次日调频容量+里程报价 | 调频实施细则3.0版 |
| **中长期交易** | 分时段签约 | 分时段交易 | 中长期实施细则3.1版（2026年5月实施） |
| **容量补偿** | 自动获得（电网侧项目） | 核定容量×200元/kW·年 | 容量补偿分配方案 |

### 3.2 ⭐ 按小时选择机制（核心规则详解）

这是浙江储能市场最重要的规则设计，与"永久二选一"完全不同：

#### 机制说明

| 维度 | 规则 | 说明 |
|------|------|------|
| **选择频率** | **按小时**，每日D-1重新申报 | 非一次性永久选择 |
| **选择方式** | 储能申报次日24小时各小时的选择意向 | 每小时可不同 |
| **调频容量上限** | **200MW**（储能调频总容量上限） | 超出部分自动转入电能量市场 |
| **入选调频时段** | 不参与该时段的电能量市场 | 储能资源互斥占用 |
| **其余时段** | 按申报的自计划曲线参与电能量市场 | 可同时参与现货套利 |
| **SOC门槛** | 若小时内未参与电能量市场且满足SOC门槛要求，可参与该小时调频市场 | 有SOC限制 |

#### 日前市场申报流程（D-1日）

```
D-1日9:30前  调度机构披露D日电网侧储能可调容量（D-1日披露，为申报提供参考）
D-1日12:00前 储能完成D日各小时选择申报（申报调频？还是申报电能量？）
D-1日16:00前 调度机构发布调频市场出清结果
             → 入选调频的储能（≤200MW）：该小时不参与电能量市场
             → 未入选或剩余储能：按自计划曲线参与电能量市场
```

#### 实际操作示例

某50MW/100MWh储能电站，D日计划：

| 时段 | 选择 | 原因 |
|------|------|------|
| 0-8时（深夜低谷） | 电能量市场（充电） | 低电价时段充电 |
| 8-12时（早高峰） | 调频市场 | 高电价+高调频需求，收益更高 |
| 12-18时（平段） | 电能量市场（放电） | 高电价放电套利 |
| 18-24时（晚高峰） | 调频市场 | 再次参与调频 |

> ⭐ **核心优势**：按小时动态调整，可同时享受调频收益和现货峰谷价差，比"永久二选一"灵活得多。
