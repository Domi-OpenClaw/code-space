// security & ops

ECS + BCF 双体系自检 — 检查→自愈→告警

🏠 ECS — 基础保障

确保 Domi 自身稳定

  • 📦 服务器：CPU / 内存 / 磁盘 / Swap
  • ⚙️ 操作系统：关键进程 / 系统日志
  • 🤖 OpenClaw：进程存活 / Cron Jobs
  • 💾 备份：今日备份完整性

💼 BCF — 业务保障

确保所有业务 Skills 稳定

  • Skills：目录存在性 + SKILL.md 完整性
  • 基础设施：API 健康检查(:10129/:10130)
  • API 守护：watchdog 自动重启
🔴 严重：ECS 失败 / API 全宕 / 磁盘>90% → 立即钉钉告警
🟡 警告：BCF 异常 / 单 API 断 → 连续 3 次通知
