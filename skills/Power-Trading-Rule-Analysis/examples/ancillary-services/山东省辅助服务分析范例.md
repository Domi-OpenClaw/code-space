## 辅助服务补偿机制

### 调频补偿公式

```
调频补偿电费 = 调频容量 × 调频补偿标准（15 元/MW）+ 调频里程 × 调频里程补偿标准
```

**示例计算**：
- 某储能电站提供调频服务，调频容量 10MW，调频里程 50MWh
- 调频补偿 = 10MW × 15 元/MW = 150 元
- 调频里程补偿（假设 5 元/MWh）= 50MWh × 5 元/MWh = 250 元
- **合计**：150 + 250 = 400 元

### 调峰补偿公式

```
调峰补偿电费 = 调峰电量 × 调峰补偿标准
```

**示例计算**：
- 某火电机组深度调峰 100MWh，补偿标准 200 元/MWh
- 调峰补偿 = 100MWh × 200 元/MWh = 20,000 元

### 备用补偿公式

```
备用补偿电费 = 备用容量 × 备用补偿标准
```
