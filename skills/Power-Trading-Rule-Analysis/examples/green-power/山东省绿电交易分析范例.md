## 绿电价格构成

### 价格公式

```
绿电价格 = 电能量价格 + 环境价值（绿证价格）
```

### 电能量价格

| 类型 | 价格范围 | 说明 | 政策依据 |
|------|---------|------|---------|
| 燃煤基准价 | 0.3949 元/kWh | 参考基准 | 鲁发改价格〔2025〕871 号 |
| 绿电交易价格 | 0.38-0.45 元/kWh | 2026 年 1 月均价 0.41 元/kWh | 鲁电交〔2026〕40 号 |

### 环境价值（绿证价格）

| 类型 | 价格范围 | 说明 | 政策依据 |
|------|---------|------|---------|
| 风电绿证 | 20-50 元/个 | 2025 年均价 35 元/个 | 发改能源〔2025〕262 号 |
| 光伏绿证 | 30-80 元/个 | 2025 年均价 55 元/个 | 发改能源〔2025〕262 号 |
| 1 个绿证 | 1000kWh 电量 | 对应 1000kWh 可再生能源电量 | 国能发新能规〔2024〕67 号 |

### 综合绿电价格示例

**假设**：某企业购买 100 万 kWh 光伏绿电，电能量价格 0.42 元/kWh，绿证价格 55 元/个（即 0.055 元/kWh）。

```
电能量费用 = 100 万 kWh × 0.42 元/kWh = 42 万元
绿证费用 = 1000 个 × 55 元/个 = 5.5 万元
合计费用 = 42 + 5.5 = 47.5 万元
综合电价 = 47.5 万元 / 100 万 kWh = 0.475 元/kWh
```

## 绿电交易流程

### 月度绿电集中竞价流程

```
每月 20 日前 → 发电企业申报（电量、价格）
       ↓
每月 23 日 9:00 → 用户/售电公司申报（电量、价格）
       ↓
每月 23 日 11:00 → 集中撮合（边际出清）
       ↓
每月 23 日 14:00 → 发布成交结果
       ↓
每月 25 日前 → 签订绿电交易合同
       ↓
次月 → 执行交易、结算
```

### 证电分离交易流程

```
步骤 1：购买常规电力（电力交易中心）
       ↓
步骤 2：购买绿证（绿证交易平台）
       ↓
步骤 3：声明使用绿电（碳排放核算）
```

**山东特色**：支持证电合一和证电分离两种模式。

## 绿电交易与碳市场衔接

### 碳排放核算规则

| 场景 | 核算方法 | 政策依据 |
|------|---------|---------|
| 使用绿电（证电合一） | 碳排放因子 = 0 | 鲁发改能源〔2025〕920 号 |
| 使用绿证（证电分离） | 碳排放因子 = 0 | 鲁发改能源〔2025〕920 号 |
| 使用常规电力 | 碳排放因子 = 电网平均排放因子 | 鲁发改能源〔2025〕920 号 |

### 出口型企业绿电需求

**背景**：欧盟碳边境调节机制（CBAM）要求出口产品披露碳排放。

**山东政策**：
- 出口型企业购买绿电，碳排放因子 = 0
- 绿电消费凭证可作为出口产品碳足迹证明
- **政策依据**：鲁发改能源〔2025〕920 号

## 新能源可持续发展机制

### 差价结算机制

**适用对象**：2025 年及以后投产的新能源项目

**结算公式**：
```
机制电量 = 新能源项目上网电量 × 机制电量比例（待明确）
机制电价 = 燃煤基准价（0.3949 元/kWh）
差价 = 机制电价 - 现货市场均价
结算电费 = 机制电量 × 差价
```

**政策依据**：鲁发改价格〔2025〕871 号

### 机制电价执行

| 项目类型 | 执行电价 | 政策依据 |
|---------|---------|---------|
| 2025 年前投产 | 原上网电价政策 | 鲁发改价格〔2025〕576 号 |
| 2025 年及以后投产 | 机制电价（燃煤基准价） | 鲁发改价格〔2025〕576 号 |
| 竞价配置项目 | 竞价上网电价 | 鲁发改价格〔2025〕576 号 |
