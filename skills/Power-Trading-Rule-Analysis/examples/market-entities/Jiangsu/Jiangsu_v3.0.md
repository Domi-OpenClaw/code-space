# 第一章 市场主体概览

## 第一节 市场主体类型

江苏省电力市场主体分为以下五大类：

| 主体类型 | 细分类型 | 准入条件概要 | 政策依据 |
|---------|---------|------------|---------|
| 发电企业 | 火电、水电、核电、新能源（风电/光伏）、独立储能 | 取得电力业务许可证，接入电网，并网运行 | 苏发改能源发〔2023〕1428号 |
| 电力用户 | 工商业用户、居民农业用户、批发用户 | 10kV及以上电压等级，年用电量达标 | 苏发改能源〔2022〕1090号 |

| 新型主体 | 独立储能、虚拟电厂、负荷聚合商 | 满足容量/调节能力/响应时间等技术要求 | 苏发改能源〔2024〕567/568号 |

## 第五节 出清机制详解

### 一、集中竞价出清流程

```
1. 申报阶段（每月15日10:00-10:50）
   ↓
2. 申报截止（10:50），系统锁定所有申报
   ↓
3. 撮合阶段（11:00-11:05）
   - 按价格优先、时间优先原则撮合
   - 买方高价优先，卖方低价优先
   ↓
4. 统一出清价格形成
   - 边际出清价格 = 最后一笔成交价格
   - 所有成交均按此价格结算
   ↓
5. 结果发布（11:05后）
   - 交易平台发布成交结果
```

**成交优先顺序**：
1. 价格优先：买方报价高者优先，卖方报价低者优先
2. 时间优先：同一价格下，申报时间早者优先

### 二、双边协商出清流程

```
1. 双方协商确定电量、价格、曲线
   ↓
2. 卖方在平台申报（电量、价格、曲线）
   ↓
3. 买方在平台确认
   ↓
4. 交易平台即时发布成交结果
   ↓
5. 签订电子合同（平台自动生成）
```

### 三、挂牌交易出清流程

```
1. 挂牌阶段（9:00-9:50）
```

## 第五节 偏差考核计算示例

### 示例1：电力用户偏差考核

**假设**：某电力用户月度合同电量100万kWh，实际用电量108万kWh，免考核范围±3%，考核系数1.2，月度均价0.42元/kWh。

```
偏差电量 = 108 - 100 = +8万kWh（超用）
偏差率 = 8/100 × 100% = 8%
免考核电量 = 100 × 3% = 3万kWh
考核电量 = 8 - 3 = 5万kWh
考核费用 = 5万kWh × 1.2 × 0.42元/kWh = 2.52万元
```

### 示例2：售电公司偏差考核

**假设**：某售电公司代理用户总合同电量1000万kWh，实际用电量920万kWh，免考核范围±3%，考核系数0.8，合约均价0.42元/kWh。

```
偏差电量 = 920 - 1000 = -80万kWh（少用）
偏差率 = -80/1000 × 100% = -8%
免考核电量 = 1000 × 3% = 30万kWh
考核电量 = 80 - 30 = 50万kWh（取绝对值）
考核费用 = 50万kWh × 0.8 × 0.42元/kWh = 16.8万元
```

### 示例3：发电企业偏差考核

**假设**：某发电企业月度合同电量5000万kWh，实际发电量4800万kWh，免考核范围±3%，考核系数0.8，标杆电价0.391元/kWh。

```
偏差电量 = 4800 - 5000 = -200万kWh（少发）
偏差率 = -200/5000 × 100% = -4%
免考核电量 = 5000 × 3% = 150万kWh
考核电量 = 200 - 150 = 50万kWh（取绝对值）
考核费用 = 50万kWh × 0.8 × 0.391元/kWh = 15.64万元
```
