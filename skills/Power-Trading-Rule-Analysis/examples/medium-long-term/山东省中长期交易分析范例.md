## 出清机制

### 集中竞价出清

**出清方式**：边际出清

**出清公式**：
```
出清价 = 最后一对成交报价的均价
出清量 = 成交电量
```

**撮合规则**：
- 价格优先：高价买入申报优先于低价买入申报
- 时间优先：同价位申报，先申报优先
- 成交原则：买入申报价 ≥ 卖出申报价 → 成交

**示例计算**：
```
买入申报：0.42 元/kWh，电量 100 万 kWh
卖出申报：0.40 元/kWh，电量 100 万 kWh
出清价 = (0.42 + 0.40) / 2 = 0.41 元/kWh
```

### 滚动撮合出清

**出清方式**：价格优先、时间优先

**撮合规则**：
- 连续撮合：实时匹配买入和卖出申报
- 成交价格：(买入价 + 卖出价) / 2
- 撮合周期：D+2/D+3 交易日

## 省份结算机制（山东 40%+60%）

### 结算公式

```
中长期合约总结算电费 = 40% × 合约电量 × 中长期价格 + 60% × 合约电量 × 现货价格
```

### 示例计算

**假设**：某售电公司月度中长期合约 100 万 kWh，中长期价格 0.42 元/kWh，现货均价 0.38 元/kWh。

```
40% 部分结算 = 40 万 kWh × 0.42 元/kWh = 16.8 万元
60% 部分结算 = 60 万 kWh × 0.38 元/kWh = 22.8 万元
合计结算 = 16.8 + 22.8 = 39.6 万元
综合电价 = 39.6 万元 / 100 万 kWh = 0.396 元/kWh
```

**山东特色**：全国首个 40%+60% 结算机制省份，中长期合约电量仅 40% 按中长期价格结算，60% 按现货价格结算。

## 售电公司购电策略

**核心逻辑**：售电公司一定会**贴着月度交易公告规定的下限**购买中长期电量。

**成本对比示例**：
- 方案 A：签约 85%（政策文件年度要求）
  - 中长期电量：85 万 kWh × 0.42 元/kWh = 35.7 万元
  - 现货电量：15 万 kWh × 0.38 元/kWh = 5.7 万元
  - 合计：41.4 万元

- 方案 B：签约 29.8%（3 月公告下限）
  - 中长期电量：29.8 万 kWh × 0.42 元/kWh = 12.516 万元
  - 现货电量：70.2 万 kWh × 0.38 元/kWh = 26.676 万元
  - 合计：39.192 万元

- **节省**：41.4 - 39.192 = 2.208 万元（5.3%）

**策略建议**：售电公司应密切关注月度交易公告，贴着下限购买中长期电量，保留更多电量参与现货市场获取波动收益。
