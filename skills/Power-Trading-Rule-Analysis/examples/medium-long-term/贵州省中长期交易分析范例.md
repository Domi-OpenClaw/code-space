## 出清机制

### 集中竞价出清

**出清方式**：边际出清

**出清公式**：
```
出清价 = 最后一对成交报价的均价
出清量 = 成交电量
```

**撮合规则**：
- 价格优先：高价买入申报优先于低价买入申报
- 时间优先：同价位申报，先申报优先
- 成交原则：买入申报价 ≥ 卖出申报价 → 成交

### 双边协商出清

**出清方式**：协商定价

**撮合规则**：
- 双方协商一致后确认
- 交易平台备案

## 省份结算机制（贵州型：中长期 + 现货偏差）

### 结算公式

```
总结算电费 = 中长期合约电量 × 中长期价格 + 偏差电量 × 现货价格
偏差电量 = 实际用电量 - 中长期合约电量
```

### 示例计算

**假设**：某售电公司月度中长期合约 80 万 kWh，中长期价格 0.40 元/kWh，实际用电量 100 万 kWh，现货均价 0.38 元/kWh。

```
中长期部分结算 = 80 万 kWh × 0.40 元/kWh = 32 万元
偏差电量 = 100 - 80 = 20 万 kWh（正偏差）
偏差部分结算 = 20 万 kWh × 0.38 元/kWh = 7.6 万元
合计结算 = 32 + 7.6 = 39.6 万元
综合电价 = 39.6 万元 / 100 万 kWh = 0.396 元/kWh
```

**贵州特色**：南方区域统一结算机制，中长期合约电量按中长期价格结算，偏差电量按现货价格结算。

## 售电公司购电策略

**核心逻辑**：售电公司一定会**贴着月度交易公告规定的下限**购买中长期电量。

**成本对比示例**：
- 方案 A：签约 85%（政策文件年度要求）
  - 中长期电量：85 万 kWh × 0.40 元/kWh = 34 万元
  - 现货电量：15 万 kWh × 0.38 元/kWh = 5.7 万元
  - 合计：39.7 万元

- 方案 B：签约 80%（年度下限）
  - 中长期电量：80 万 kWh × 0.40 元/kWh = 32 万元
  - 现货电量：20 万 kWh × 0.38 元/kWh = 7.6 万元
  - 合计：39.6 万元

- **节省**：39.7 - 39.6 = 0.1 万元（0.25%）

**策略建议**：贵州省现货价格波动较小，售电公司购电策略相对保守，建议按年度下限 80% 签约。
