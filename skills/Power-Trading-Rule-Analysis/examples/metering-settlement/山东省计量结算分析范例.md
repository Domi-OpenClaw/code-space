## 结算公式

### 发电侧结算

```
发电侧总结算电费 = 日前市场电费 + 实时市场电费 + 中长期合约电费 + 辅助服务电费 + 容量补偿电费 + 其他费用

日前市场电费 = ∑(各时段日前电量 × 日前节点电价)
实时市场电费 = ∑(各时段实时偏差电量 × 实时节点电价)
中长期合约电费 = 40% × 合约电量 × 中长期价格 + 60% × 合约电量 × 现货均价
辅助服务电费 = 调频电费 + 调峰电费 + 备用服务费
容量补偿电费 = 机组容量 × 容量补偿标准（0.096 元/kWh）
```

### 用户侧结算

```
用户侧总结算电费 = 电能量电费 + 输配电费 + 政府性基金及附加 + 辅助服务分摊 + 容量补偿分摊

电能量电费 = ∑(各时段用电量 × 统一结算点电价)
输配电费 = 用电量 × 输配电价
政府性基金及附加 = 用电量 × 政府性基金标准
辅助服务分摊 = 按用电量比例分摊
容量补偿分摊 = 按用电量比例分摊
```

### 售电公司结算

```
售电公司总结算电费 = 批发侧购电费 + 零售侧售电费 + 偏差考核电费 + 其他费用

批发侧购电费 = 中长期合约电费 + 现货市场购电费
零售侧售电费 = ∑(代理用户用电量 × 零售套餐价格)
偏差考核电费 = 偏差电量 × 偏差考核价格
```

## 山东 40%+60% 结算机制

### 结算公式详解

```
中长期合约总结算电费 = 40% × 合约电量 × 中长期价格 + 60% × 合约电量 × 现货均价
```

### 示例计算

**假设**：某售电公司月度中长期合约 100 万 kWh，中长期价格 0.42 元/kWh，现货均价 0.38 元/kWh。

```
40% 部分结算 = 40 万 kWh × 0.42 元/kWh = 16.8 万元
60% 部分结算 = 60 万 kWh × 0.38 元/kWh = 22.8 万元
合计结算 = 16.8 + 22.8 = 39.6 万元
综合电价 = 39.6 万元 / 100 万 kWh = 0.396 元/kWh
```

**山东特色**：全国首个 40%+60% 结算机制省份，中长期合约电量仅 40% 按中长期价格结算，60% 按现货价格结算。

## 新能源差价结算

### 结算公式

```
机制电量 = 新能源项目上网电量 × 机制电量比例（待明确）
机制电价 = 燃煤基准价（0.3949 元/kWh）
差价 = 机制电价 - 现货市场均价
结算电费 = 机制电量 × 差价
```

### 示例计算

**假设**：某光伏电站月上网电量 100 万 kWh，机制电量比例 80%，现货均价 0.35 元/kWh。

```
机制电量 = 100 万 kWh × 80% = 80 万 kWh
差价 = 0.3949 - 0.35 = 0.0449 元/kWh
结算电费 = 80 万 kWh × 0.0449 元/kWh = 3.592 万元
```

**政策依据**：鲁发改价格〔2025〕871 号
