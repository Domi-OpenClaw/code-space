## 结算公式

### 发电侧结算

```
发电侧总结算电费 = 中长期合约电费 + 现货市场电费（模拟） + 辅助服务电费

中长期合约电费 = ∑(合约电量 × 中长期价格)
现货市场电费 = ∑(偏差电量 × 现货价格)（模拟）
辅助服务电费 = 调频电费 + 调峰电费
```

### 用户侧结算

```
用户侧总结算电费 = 电能量电费 + 输配电费 + 政府性基金及附加 + 辅助服务分摊

电能量电费 = ∑(各时段用电量 × 统一结算点电价)
输配电费 = 用电量 × 输配电价
政府性基金及附加 = 用电量 × 政府性基金标准
辅助服务分摊 = 按用电量比例分摊
```

### 售电公司结算

```
售电公司总结算电费 = 批发侧购电费 + 零售侧售电费 + 偏差考核电费

批发侧购电费 = 中长期合约电费 + 现货市场购电费（模拟）
零售侧售电费 = ∑(代理用户用电量 × 零售套餐价格)
偏差考核电费 = 偏差电量 × 偏差考核价格
```

## 贵州结算机制（中长期 + 现货偏差）

### 结算公式详解

```
总结算电费 = 中长期合约电量 × 中长期价格 + 偏差电量 × 现货价格
偏差电量 = 实际用电量 - 中长期合约电量
```

### 示例计算

**假设**：某售电公司月度中长期合约 80 万 kWh，中长期价格 0.40 元/kWh，实际用电量 100 万 kWh，现货均价 0.38 元/kWh。

```
中长期部分结算 = 80 万 kWh × 0.40 元/kWh = 32 万元
偏差电量 = 100 - 80 = 20 万 kWh（正偏差）
偏差部分结算 = 20 万 kWh × 0.38 元/kWh = 7.6 万元
合计结算 = 32 + 7.6 = 39.6 万元
综合电价 = 39.6 万元 / 100 万 kWh = 0.396 元/kWh
```

**贵州特色**：南方区域统一结算机制，中长期合约电量按中长期价格结算，偏差电量按现货价格结算。
