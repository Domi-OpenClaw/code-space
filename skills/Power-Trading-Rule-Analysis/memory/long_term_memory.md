## 2026-03-20: 零售套餐多维分类方法论

### 山东 6 维分类框架（用户确认）

零售套餐分类**不是固定数字**，是**多维交叉分类系统**：

| 维度 | 类型 | 关键规则 |
|------|------|---------|
| 1. 价格形成机制 | 固定/分时/价格联动/价差分享/混合 | 参考价格联动和现货联动的价格联动类型免偏差考核 |
| 2. 能源类型 | 常规/绿电（证电分离）/新能源 | 绿电=电能量价格+绿证价格 |
| 3. 交易周期 | 年度/季度/月度/月内 | 山东 2026 Q2：年度未组织，零售合同按季度 |
| 4. 用户类型 | 工商业/居民/农业/特殊用户 | 商业综合体/物业/写字楼须选价格封顶或上传商户授权 |
| 5. 特殊机制 | 价格封顶（0.006元/kWh）/需求响应/柔性调节（VPP） | 所有套餐类型须提供价格封顶选项 |
| 6. 结算方式 | 节点电价/统一结算点 | — |

**价格封顶机制**（山东创新）：浮动值 0.006 元/kWh（含税），不选需法人人脸识别+48h 冷静期+知情同意书。

**售电公司披露要求**：须发布≥2 种不同类型标准化套餐，必须包含≥1 个价格联动套餐。

### 贵州 5 维分类框架

| 维度 | 关键差异（对比山东） |
|------|-------------------|
| 价格机制 | 固定/分时（1.6:1:0.4）/市场联动（待细则） |
| 能源类型 | 常规/绿电（证电分离） |
| 交易周期 | 年度（≥80%签约）/月度/月内 |
| 用户类型 | 10kV 及以下须通过售电公司 |
| 特殊机制 | 默认零售价 1.5 倍代理购电价 |

**贵州市场风险**（2026年1月）：批零倒挂，售电市场整体亏损约 3000 万元，批发均价 0.3796 元/kWh，零售均价 0.3754 元/kWh。

**省间对比要点**：禁止跨省套用分类方式，每省按实际政策呈现。
