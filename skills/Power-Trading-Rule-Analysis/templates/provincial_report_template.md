```
{主体类型}收益 = 电能量交易收益（放电收益 - 充电成本）+ 辅助服务收益 + 容量补偿 - 相关费用
```

|---------|---------|---------|--------------|
| 调频服务 | 调频补偿价格 | X-X 元/MWh | ... |
| | Kp 值（性能指标） | X-X | ... |
| | 调频里程 | ... | ... |
| 调峰服务 | 调峰补偿价格 | X-X 元/kWh | ... |
| | 填谷补贴标准 | X-X 元/kWh | ... |
| 容量补偿 | 容量电价 | X-X 元/kW·月 | ... |
| | 可用率考核阈值 | ≥X% | ... |
| 备用服务 | 旋转备用补偿 | X-X 元/MW·h | ... |
| | 非旋转备用补偿 | X-X 元/MW·h | ... |
| 需求响应 | 邀约型补贴 | X-X 元/kWh | ... |
| | 竞价型补贴 | ... | ... |
| 电能量交易 | 现货价格 | X-X 元/kWh | ... |
| | 中长期价格 | X-X 元/kWh | ... |

**参数说明**：
1. 调频收益 = 调频里程 × 调频补偿价格 × Kp 值
2. 调峰收益 = 调峰电量 × 调峰补偿价格
3. 容量补偿 = 核定容量 × 容量电价 × 月度可用率
4. 现货价格波动范围大，新能源大发时段可能出现负电价
5. 以上参数为参考范围，具体以当地电力交易中心、能监办最新文件为准
