# Deviation Assessment Analyzer Tool

**Tool 名称**: 偏差考核分析工具  
**版本**: V1.0.0  
**创建日期**: 2026-03-20  
**所属 Skill**: Power-Trading-Rule-Analysis  
**文件位置**: `tools/deviation-assessment-analyzer.md`

---

## Tool 定位

独立分析各省份电力偏差考核规则，输出考核标准、豁免条件、修正机制、预警机制等内容。

**适用场景**：
- 生成省级电力交易规则分析报告的"偏差考核与修正"章节
- 单独查询某省偏差考核规则
- 跨省偏差考核规则对比分析
- 售电公司偏差风险管理

---

## 核心原则（最高优先级）

### 数据来源真实性原则
- ❌ 严禁填写未实际获取的考核数据
- ❌ 严禁编造豁免条件、考核系数
- ✅ 如无法获取官方文件，明确标注"数据待补充"
- ✅ 核心参数必须搜索确认，但需标注来源

### 省份差异原则
- 不同省份偏差考核标准不同（±2%、±3%、±5%）
- 不同省份豁免条件不同（价格联动类免考核等）
- 不同省份考核系数不同
- 不能假设所有省份规则相同

---

## 分析流程（强制 5 步）

### Step 1: 政策文件搜索

**搜索关键词**：
- `{省份} 电力市场 偏差考核 实施细则`
- `{省份} 电力中长期 偏差管理`
- `{省份} 售电公司 偏差考核`
- `{省份} 电力市场 豁免条件`
- `{省份} 电力市场 预警机制`

**目标文件**：
- 电力中长期交易偏差考核实施细则
- 电力市场偏差管理办法
- 售电公司偏差考核实施细则
- 电力市场风险预警机制

### Step 2: 考核标准提取

**偏差定义**：
- 实际用电量 vs 合同电量
- 实际发电量 vs 合同电量

**免考核范围**：
- ±2%、±3%、±5% 等

**考核系数**：
- 超用考核系数
- 少用考核系数

### Step 3: 豁免条件提取

**豁免类型**：
- 价格联动类套餐免考核（山东）
- 政策性因素豁免
- 不可抗力豁免
- 电网原因豁免

**豁免申请流程**：
- 申请材料
- 审批流程
- 审批时限

### Step 4: 修正机制提取

**修正类型**：
- 合同电量转让
- 合同电量置换
- 偏差电量购买

**修正流程**：
- 申请时间
- 交易平台申报
- 成交确认

### Step 5: 预警机制提取

**预警等级**：
- 一级预警（红色）
- 二级预警（橙色）
- 三级预警（黄色）
- 四级预警（蓝色）

**预警阈值**：
- 各等级阈值
- 触发条件

**预警响应**：
- 各等级响应措施

## 第 X 节 偏差考核计算示例

### 示例 1：电力用户偏差考核

**假设**：某电力用户月度合同电量 100 万 kWh，实际用电量 108 万 kWh，免考核范围±5%，考核系数 1.2，月度集中竞价均价 0.42 元/kWh。

```
偏差电量 = 108 - 100 = 8 万 kWh
偏差率 = 8/100 × 100% = 8%
免考核电量 = 100 × 5% = 5 万 kWh
考核电量 = 8 - 5 = 3 万 kWh
考核费用 = 3 万 kWh × 1.2 × 0.42 元/kWh = 1.512 万元
```

### 示例 2：售电公司偏差考核（价格联动类免考核）

**假设**：某售电公司代理用户总合同电量 1000 万 kWh，实际用电量 920 万 kWh，其中 600 万 kWh 签订价格联动套餐。

```
总偏差电量 = 920 - 1000 = -80 万 kWh
价格联动类免考核电量 = 600 万 kWh 对应偏差
剩余考核电量 = -80 万 kWh × (400/1000) = -32 万 kWh
偏差率 = -32/400 × 100% = -8%
免考核范围：±5%
考核电量 = -32 万 kWh - (400 × 5%) = -12 万 kWh
考核费用 = 12 万 kWh × 考核系数 × 价格 = {数值} 万元
```

### 示例 3：发电企业偏差考核

**假设**：某发电企业月度合同电量 5000 万 kWh，实际发电量 4800 万 kWh，免考核范围±3%，考核系数 0.8，标杆电价 0.4229 元/kWh。

```
偏差电量 = 4800 - 5000 = -200 万 kWh
偏差率 = -200/5000 × 100% = -4%
免考核电量 = 5000 × 3% = 150 万 kWh
考核电量 = 200 - 150 = 50 万 kWh
考核费用 = 50 万 kWh × 0.8 × 0.4229 元/kWh = 16.916 万元
```
