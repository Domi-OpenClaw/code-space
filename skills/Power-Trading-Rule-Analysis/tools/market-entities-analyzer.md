# Market Entities Analyzer Tool

**Tool 名称**: 市场主体分析工具  
**版本**: V1.0.0  
**创建日期**: 2026-03-20  
**所属 Skill**: Power-Trading-Rule-Analysis  
**文件位置**: `tools/market-entities-analyzer.md`

---

## Tool 定位

独立分析各省份电力市场主体规则，输出准入条件、注册流程、退出机制、主体权利义务等内容。

**适用场景**：
- 生成省级电力交易规则分析报告的"市场主体概览"章节
- 单独查询某省市场主体准入规则
- 跨省市场主体规则对比分析
- 售电公司/新能源企业入市咨询

---

## 核心原则（最高优先级）

### 数据来源真实性原则
- ❌ 严禁填写未实际获取的政策文件
- ❌ 严禁编造政策文件文号
- ✅ 如无法获取官方文件，明确标注"数据待补充"
- ✅ 核心参数必须搜索确认，但需标注来源

### 省份差异原则
- 不同省份市场主体准入条件不同（电压等级、用电量门槛等）
- 不同省份注册流程不同（线上/线下、审批时限等）
- 不能假设所有省份规则相同

## 分析流程（强制 5 步）

### Step 1: 政策文件搜索

**搜索关键词**：
- `{省份} 电力市场主体 注册管理办法`
- `{省份} 售电公司 准入与退出`
- `{省份} 电力用户 入市条件`
- `{省份} 发电企业 市场准入`
- `{省份} 电力交易中心 注册流程`

**目标文件**：
- 电力市场主体注册管理办法
- 售电公司准入与退出管理实施细则
- 电力用户入市交易管理办法
- 发电企业参与电力市场交易实施细则
- 新型经营主体（储能、虚拟电厂）管理细则

### Step 2: 主体类型提取

提取该省允许参与市场的主体类型：

| 主体类型 | 提取内容 | 输出格式 |
|---------|---------|---------|
| 发电企业 | 火电、水电、核电、新能源、储能等 | 列表 + 准入条件 |
| 电力用户 | 工商业、居民、农业等 | 列表 + 准入条件 |
| 售电公司 | 独立售电、电网企业售电等 | 列表 + 准入条件 |
| 新型主体 | 独立储能、虚拟电厂、负荷聚合商等 | 列表 + 准入条件 |
| 电网企业 | 输配电、调度、结算等 | 列表 + 职能说明 |

### Step 3: 准入条件详情提取

**发电企业**：
- 装机容量门槛
- 电压等级要求
- 许可证要求（电力业务许可证）
- 环保要求（超低排放、环评等）
- 并网要求

**电力用户**：
- 电压等级门槛（如 10kV 及以上）
- 用电量门槛（如年用电量 500 万千瓦时）
- 行业限制（高耗能、淘汰类等）
- 计量要求

**售电公司**：
- 注册资本要求
- 从业人员要求
- 经营场所要求
- 技术支持系统要求
- 履约保函/保险要求

**新型主体**：
- 独立储能：容量要求、充放电效率、响应时间等
- 虚拟电厂：聚合资源类型、可调节能力、通信要求等
- 负荷聚合商：聚合用户数量、可中断负荷容量等

### Step 4: 注册流程提取

| 流程环节 | 提取内容 | 输出格式 |
|---------|---------|---------|
| 注册平台 | 线上平台名称/线下办理地点 | 文字说明 |
| 提交材料 | 所需材料清单 | 列表 |
| 审核时限 | 审批工作日 | 数值 |
| 公示要求 | 公示平台、公示期 | 文字说明 |
| 生效条件 | 生效时间、生效标志 | 文字说明 |

### Step 5: 退出机制提取

**退出类型**：
- 自愿退出（申请流程、条件限制）
- 强制退出（违规情形、处罚措施）
- 保底承接（保底售电公司、承接流程）

**退出后果**：
- 履约保函处理
- 欠费清缴
- 信用影响
- 重新入市限制

## 检查清单（生成后强制执行）

### 政策文号检查
- [ ] 每个主体类型都有政策文号支撑
- [ ] 准入条件有政策文号
- [ ] 注册流程有政策文号
- [ ] 退出机制有政策文号
- [ ] 政策文件数量 ≥ 5 个

### 内容完整性检查
- [ ] 5 类主体类型完整呈现（发电、用户、售电、新型、电网）
- [ ] 准入条件详解（分主体类型）
- [ ] 注册流程完整（平台、材料、时限、公示、生效）
- [ ] 退出机制完整（自愿、强制、保底）
- [ ] 权利义务说明

### 省份差异检查
- [ ] 未套用其他省份准入条件
- [ ] 按该省实际政策文件呈现
- [ ] 查不到的内容标注"待补充"或"待细则"

### 数据真实性检查
- [ ] 未捏造政策文件文号
- [ ] 未实际获取的数据来源标注"数据待补充"
- [ ] 核心参数已搜索确认并标注来源

## 范例库

### 山东省范例

**政策文件**：
- 《山东电力市场规则（试行）》（鲁监能市场规〔2025〕57 号）
- 《山东省售电公司准入与退出管理实施细则》
- 《山东省电力用户入市交易管理办法》
- 《山东省独立储能参与电力市场交易实施细则》
- 《山东省虚拟电厂参与电力市场交易实施细则》

**准入条件特色**：
- 电力用户：不受电压等级和用电量限制
- 售电公司：注册资本≥2000 万元，10 名以上专业人员
- 独立储能：容量≥10MW，充放电效率≥85%
- 虚拟电厂：可调节能力≥1MW，响应时间≤15 分钟

**范例文件路径**：`examples/market-entities/山东省市场主体分析范例.md`
