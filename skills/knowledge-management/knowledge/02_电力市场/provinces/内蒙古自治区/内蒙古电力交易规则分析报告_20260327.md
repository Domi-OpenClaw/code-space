# 内蒙古电力交易规则分析报告

**报告日期：2026年3月27日**
**对标标准：山东省电力市场规则V13.0**
**适用范围：蒙西电网（内蒙古电力多边交易市场）+ 蒙东电网（内蒙古东部电力市场）**

---

## 摘要

内蒙古是全国新能源装机第一大省，风光资源禀赋突出，新能源装机规模连续多年位居全国首位。截至2025年底，内蒙古新能源装机已超过1.6亿千瓦，占全区电源装机比重超过60%。内蒙古电网分为蒙西电网（内蒙古电力集团管辖）和蒙东电网（国网蒙东电力管辖）两大独立省级电网，其中蒙西电网于2025年2月24日正式转入电力现货市场正式运行，成为全国第5个转入正式运行的电力现货市场；蒙东电网于2025年6月启动现货市场试运行，并纳入东北电网统一调度。

2024年12月27日，内蒙古自治区能源局联合国家能源局华北监管局正式印发《内蒙古电力多边交易市场规则体系》（"1+8"系列规则），自2025年1月1日起施行，标志着内蒙古电力市场规则体系进入全新阶段。2025年10月17日，蒙东电力市场规则体系（"1+8"系列规则）正式印发。两个规则体系在市场主体准入、交易机制、结算方式等方面存在显著差异。

本报告以内蒙古电力市场规则体系为核心，对标山东省V13.0标准，从市场主体、中长期交易、现货市场、零售市场、绿电交易、计量结算、偏差考核、辅助服务、输配电价、市场风险监管等十个维度进行系统分析，并重点编制附录B关键参数汇总表（11类参数、≥100项），供产品经理、售前、销售及电力行业研究者参考使用。

## 第一章 市场主体概览

### 1.1 电网结构与市场划分

内蒙古电力市场按照电网营业范围划分为两个相对独立的省级电力市场：

**（1）蒙西电网（内蒙古电力多边交易市场）**

- 管辖范围：呼和浩特市、包头市、锡林郭勒盟、鄂尔多斯市、乌兰察布市、巴彦淖尔市、乌海市、阿拉善盟、二连浩特市
- 电网企业：内蒙古电力（集团）有限责任公司
- 市场特点：独立省级电网，与国家电网华北网局相连但独立运营
- 运营机构：内蒙古电力交易中心有限公司
- 现货市场状态：**2025年2月24日正式转入电力现货市场正式运行**（全国第5个转入正式运行的现货市场）
- 2024年新能源交易电量占比：92%
- 2025年区内电力市场交易电量规模预计：约2900亿千瓦时（含居民农业用电254亿千瓦时）

**（2）蒙东电网（内蒙古东部电力市场）**

- 管辖范围：呼伦贝尔市、兴安盟、通辽市、赤峰市
- 电网企业：国网内蒙古东部电力有限公司（国家电网全资子公司）
- 市场特点：与东北电网（辽宁、吉林、黑龙江）一体化运行
- 运营机构：内蒙古东部电力交易中心
- 现货市场状态：2025年6月启动现货市场试运行
- 2026年区内电力市场交易电量规模预计：约533亿千瓦时（含工商业线损），直接交易251亿千瓦时，电网公司代理交易100亿千瓦时

### 1.2 市场主体分类

根据《内蒙古电力多边交易市场基本规则》和《蒙东电力市场规则体系（试行）》，内蒙古电力市场主体主要包括以下类别：

**发电企业类**

| 主体类型 | 蒙西电网 | 蒙东电网 | 准入条件 |
|---------|---------|---------|---------|
| 燃煤发电企业 | ✓ | ✓ | 具备电力业务许可证，纳入电网调度管理 |
| 风电企业 | ✓ | ✓ | 具备电力业务许可证，环保设施达标 |
| 光伏企业 | ✓ | ✓ | 具备电力业务许可证 |
| 分布式新能源 | ✓ | ✓ | 通过虚拟电厂聚合参与 |
| 储能放电（双重身份） | ✓ | ✓ | 放电时作为发电企业，充电时视同电力用户 |

**电力用户类**

| 主体类型 | 蒙西电网 | 蒙东电网 | 准入条件 |
|---------|---------|---------|---------|
| 直接参与批发用户 | ✓ | ✓ | 10kV及以上工商业用户（含限制类）原则上全部直接参与市场交易 |
| 电网代理购电用户 | ✓ | ✓ | 居民、农业等优先用户 |
| 零售用户 | ✓ | ✓ | 通过售电公司代理参与市场 |
| 清洁供暖用户 | ✓ | ✓ | 执行工商业分时电价 |

**售电公司**

- 须在电力交易平台完成市场注册
- 须向电力交易机构及时、足额缴纳履约保函或履约保险
- 2026年起取消"承诺价格+价差返还"和"按价格比例分摊"模式
- 新增"固定度电服务费"和"基准服务费+浮动服务费"两种模式

**新型主体**

| 主体类型 | 蒙西电网 | 蒙东电网 | 说明 |
|---------|---------|---------|------|
| 独立储能 | ✓ | ✓ | 可"双重身份"参与，放电为发电、充电为用户 |
| 虚拟电厂（负荷型） | ✓ | ✓ | 聚合独立储能+市场化用户+分布式电源 |
| 虚拟电厂（电源型） | ✓ | ✓ | 聚合分布式光伏+分散式风电 |
| 绿电直连项目 | ✓ | ✓ | 用户与新能源项目直接挂钩 |

### 1.3 蒙西与蒙东核心差异一览

| 对比维度 | 蒙西电网（多边市场） | 蒙东电网（东部市场） |
|---------|------------------|------------------|
| 电网归属 | 内蒙古电力集团 | 国家电网（东北分部） |
| 与国网关系 | 独立省级电网 | 东北电网一部分 |
| 现货市场状态 | 2025年2月正式运行（全国第5个） | 2025年6月试运行 |
| 2026年交易规模（预计） | ~2900亿kWh | ~533亿kWh |
| 监管机构 | 国家能源局华北监管局 | 国家能源局东北监管局 |
| 新能源参与方式 | 集中式+分布式+独立储能 | 集中式为主，逐步推进 |
| 辅助服务市场 | 现货规则中预留调频/备用接口 | 已建立调频辅助服务市场 |
| 分时电价政策 | 峰谷系数：上浮68%/下浮52% | 峰谷系数：参照东北电网标准 |
| 价格限价（现货） | 上限1500元/MWh，下限-50元/MWh | 上限1500元/kWh，下限-0.05元/kWh |
| 容量补偿标准（储能） | 2025年0.35元/kWh，2026年0.28元/kWh | 同蒙西 |

## 第二章 中长期交易规则

### 2.1 交易周期与组织方式

内蒙古中长期交易按"年度+月度+月内"三个周期组织，交易按工作日连续开展。

**（1）年度交易**

- 燃煤发电企业：年度中长期合约电量应不低于上一年度上网电量或本年度申报发电能力（扣除机制电量后，二者取较大值）的**60%**
- 市场化电力用户：年度交易电量原则上不低于上年度用网电量的**65%**（蒙东）
- 蒙东地区：电力用户（售电公司）与蒙东调管火电、新能源交易按**集中竞价**和**滚动撮合**方式开展；绿电交易、电力用户与分部调管火电按**双边协商**方式开展

**（2）月度交易**

- 月度及以上电力中长期合同签约电量比例不低于预计上网电量或申报发电能力（扣除机制电量后，二者取较大值）的**80%**
- 通过后续合同签订保障该比例要求

**（3）月内交易**

- 允许供需双方结合新能源出力特点，合理确定中长期合同的量价、曲线等内容
- 可根据实际灵活调整

**交易方式**

| 方式 | 说明 |
|------|------|
| 集中竞价 | 双向报价、集中撮合 |
| 双边协商 | 双方协商确定量价曲线 |
| 挂牌 | 一方挂单、对方摘牌 |
| 滚动撮合 | 按时间优先、价格优先撮合 |

### 2.2 分时签约规则

2025年蒙西电力市场进一步优化了用户侧签约分时价格的上下限规则：

- 中长期交易按日划分24小时时段
- 发用两侧经营主体根据发电特性和用电需求，结合电力供需形势和现货市场价格预期，合理参与中长期分时段交易
- 设置中长期交易价格申报上下限（经市场管理委员会审议通过）
- 目的：把控新能源出力富集时段的极端低价交易行为，维护市场秩序

### 2.3 新能源交易机制（蒙西）

根据《深化蒙西电网新能源上网电价市场化改革实施方案》（2025年7月1日起实施）：

**存量项目（2025年6月1日以前投产）**

| 项目类型 | 机制电价 | 机制电量比例 |
|---------|---------|------------|
| 补贴项目 | 0.25元/千瓦时 | 上网电量×30% |
| 平价项目 | 0.262元/千瓦时（蒙西煤电基准价） | 上网电量×50% |

**增量项目（2025年6月1日及以后投产）**

- 机制电价通过竞价形成
- 机制电量比例暂为上网电量的50%

**蒙东地区**

- 2025年初步安排常规风电"保量保价"优先发电电量69亿千瓦时（折算利用小时数390小时）
- 特许权项目16亿千瓦时（折算利用小时数2000小时），按蒙西地区燃煤基准价收购
- 低价项目2000小时以内电量按竞价价格执行
- 除上述电量外风电项目所发电量均参与电力市场

### 2.4 蒙东中长期交易安排（2026年）

根据《蒙东电力市场规则体系（试行）》2026年相关通知：

- 年度交易、月度交易和月内交易三个周期
- 集中竞价、滚动撮合、双边协商、挂牌等方式开展
- 售电公司服务费收取模式调整：新增固定度电服务费、基准服务费+浮动服务费模式
- 新型主体（独立储能、虚拟电厂等）须在蒙东电力交易平台完成市场注册

## 第三章 现货市场规则

### 3.1 蒙西电力现货市场（正式运行）

**基本信息**

- 2025年2月24日，蒙西电力现货市场在经过32个月连续结算试运行后，正式转入商业运行
- 是全国第5个转入正式运行的电力现货市场
- 2024年市场交易新能源电量占比已达92%
- 2025年蒙西电网新能源日均出清电量超过3.5亿千瓦时
- 11月份累计出清新能源电量143.85亿千瓦时，占总出清电量49%

**出清机制**

- 采用节点电价机制，体现中长期交易和现货交易的时间属性和空间属性
- 每一个节点作为独立的交易位置开展交易
- 市场化发电企业以报量报价方式参与现货市场
- 市场化用户以不报量不报价方式、作为价格接受者参与现货市场
- 独立新型储能、虚拟电厂等新型主体初期申报运行曲线、作为价格接受者参与现货市场，逐步实现报量报价参与

**价格上下限**

| 参数 | 数值 | 说明 |
|------|------|------|
| 申报价格上限 | **1,500元/MWh**（1.5元/千瓦时） | 暂定 |
| 申报价格下限 | **-50元/MWh**（-0.05元/千瓦时） | 2025年从0调整至-50 |
| 出清价格上下限 | 0-1,000元/MWh（初期） | 试运行阶段 |

**市场运行特点**

- 峰谷价差大：日最大峰谷价差曾达1,549.86元/MWh
- 高峰时段平均出清电价约0.64元/千瓦时（较燃煤基准价上浮126.4%）
- 发电侧最高出清价格达1.71元/千瓦时（较燃煤基准价上涨504.5%）
- 新能源占比超过55%后，出力波动对现货价格形成强烈冲击
- 午间光伏大发时段可能出现负电价（2025年11月已出现2天全天均价负电价）

### 3.2 蒙东电力现货市场（试运行）

**基本信息**

- 2025年6月，蒙东电力现货市场启动试运行
- 与东北电网统一调度
- 推动全部新能源电量参与现货市场中的实时市场

**价格上下限**

| 参数 | 数值 | 说明 |
|------|------|------|
| 申报价格上限 | **1.5元/千瓦时** | 暂定 |
| 申报价格下限 | **-0.05元/千瓦时** | 考虑新能源其他收益等因素 |

**衔接机制**

- 2025年6月前投产的新能源存量项目，电量规模衔接目前具有保障性质的上网电量
- 后续结合市场价格运行实际，适时评估调整现货市场申报限价

### 3.3 现货市场报价机制

**燃煤机组**

- 须申报报价曲线（分段报价）
- 报价上下限受监管
- 启动费用报价按冷态、温态、热态三档

**独立储能（蒙西）**

- "报量不报价"参与日前现货市场，优先出清，不参与市场定价
- 日内应执行日前充放电计划曲线
- 实时现货市场中优先出清，不参与市场定价
- 参与调频市场时实时跟踪调频指令

**虚拟电厂（蒙西）**

- 负荷型虚拟电厂：初期以报量不报价方式参与
- 电源型虚拟电厂：初期以报量不报价方式参与
- 逐步过渡到报量报价参与

## 第四章 零售市场规则

### 4.1 零售市场准入与运营

**准入条件**

- 10千伏及以上工商业用户（含限制类）原则上全部直接参与市场交易
- 居民、农业等用户由电网企业代理购电
- 零售用户须通过售电公司在电力交易平台签订零售合同

**售电公司服务费模式（2026年起重大变化）**

| 模式 | 说明 | 备注 |
|------|------|------|
| 固定度电服务费 | 按实际用电量收取固定服务
费费 | 新模式 |
| 基准服务费+浮动服务费 | 保底服务费+与市场挂钩的浮动费 | 新模式 |
| 承诺价格+价差返还 | —— | **取消** |
| 按价格比例分摊 | —— | **取消** |

**零售合同管理**

- 有效期须至2026年底
- 须通过电力交易平台签订
- 须向电力交易机构及时、足额缴纳履约保函或履约保险
- 须持续满足注册条件，规范参与批发和零售市场交易

### 4.2 零售市场分时电价衔接

根据2025年12月30日内蒙古电力多边交易市场2026年中长期交易《通知》：

- **直接参与市场用户（含零售用户）不再执行峰谷分时电价政策**
- 电网代理购电用户：继续执行峰谷分时电价政策，执行峰谷分时电价产生盈亏费用由电网代理购电用户分摊或分享
- 直接参与市场的用户，分时电价通过中长期合同约定

### 4.3 偏差分级考核（零售用户）

**考核机制**

- 售电公司对零售用户的偏差电量承担连带考核责任
- 偏差电量=实际用电量-合同约定电量
- 偏差结算费用按年度中长期交易平均成交价格的20%计算

**免考核范围**

- 非自身原因产生的偏差电量免考核（战略性新兴产业、电网检修、政策调整等）
- 虚拟电厂日前响应、实时调控型交易单元经认定有常态化调节能力的，调节能力内不纳入签约比例要求

### 4.4 蒙西零售市场特点

| 特征 | 说明 |
|------|------|
| 分时签约 | 用户侧签约分时价格设上下限 |
| 套餐模式 | 售电公司服务费套餐化管理 |
| 履约保函 | 须足额缴纳履约保函或履约保险 |
| 信息披露 | 电力交易机构加强零售市场运营管理 |

## 第五章 绿电交易规则

### 5.1 绿电交易机制

内蒙古作为全国新能源装机第一大省，绿电交易体系较为完善：

**交易品种**

- 绿色电力交易（对应绿证）
- 绿电直连项目交易
- 分布式新能源交易（通过虚拟电厂聚合）

**价格构成**

- 电能量价格：由市场交易形成
- 绿证价格：单独明确

**完善绿色电力交易政策**

- 申报和成交价格应分别明确电能量价格和相应绿证价格
- 完善绿证交易政策，促进绿证价格发现

### 5.2 绿证交易情况

**2025年全国绿证市场特点**

- 2025年1-12月，国家能源局共计核发绿证29.47亿个，其中可交易绿证18.93亿个

- 电量生产年为2025年的绿证交易平均价格：**5.15元/个**（同比暴涨359%）
- 光伏绿证价格暴涨459%

**内蒙古绿电交易试点**

- 2025年生产绿证销售额突破1600万元

### 5.3 绿电直连项目

## 第六章 计量、结算与费用

### 6.1 计量管理

**基本原则**

- 电能量交易采用节点电价机制
- 每一个节点作为独立的交易位置开展交易
- 分布式能源、储能等资源通过虚拟电厂聚合后作为一个交易单元参与

**计量装置要求**

- 绿电直连项目：须具备分表计量条件，发电/厂用电/自发自用/储能各单元分别安装计量装置
- 虚拟电厂：各聚合资源须在电力交易平台正式注册，并安装分项计量装置
- 储能：充放电须分别计量，以"双重身份"分别结算

### 6.2 结算机制

**中长期结算机制**

- 采用"市场均价±差价"的动态结算模式
- 差价=中长期合同价格-市场均价
- 偏差电量按偏差结算费用处理

**结算公式**

- 发电企业结算收入=中长期合同电费+现货市场电费+辅助服务费用
- 电力用户结算支出=中长期合同电费+现货市场电费+偏差结算费用
- 独立储能结算=放电收入-充电成本+容量补偿+调峰补偿

**蒙西结算特点**

- 连续结算试运行期间消除了价格形成的时间断层
- 电价能够实时反映电力供需变化
- 2025年4月，蒙西放宽现货市场限价（申报下限从0调整为-50元/MWh）

### 6.3 结算单出具与资金支付

| 项目 | 蒙西电网 | 蒙东电网 |
|------|---------|---------|
| 结算周期 | 日清月结 | 日清月结 |
| 结算单出具时间 | 月度结算完成后5个工作日内 | 月度结算完成后5个工作日内 |
| 资金支付时间 | 结算结果确认后10个工作日内 | 结算结果确认后10个工作日内 |

### 6.4 运行成本补偿

**独立储能容量补偿**

- 纳入自治区规划的独立新型储能电站享受容量补偿
- 2025年补偿标准：**0.35元/千瓦时**
- 2026年补偿标准：**0.28元/千瓦时**（降幅20%）
- 补偿期：10年（按投产年度标准执行）
- 补偿方式：按放电量计算，根据实际调用电量由未提供有偿调峰服务的市场主体分摊

**煤电容量补偿（2026年调整）**

- 自2026年1月1日起，合规在运的公用煤电机组容量电价调整为**165元/千瓦·年（含税）**
- 固定成本回收比例由30%调整至50%
- 计算公式：容量电费=机组额定容量×165元/千瓦·年

### 6.5 阻塞费用

- 采用节点电价机制后，不同节点间的价格差异形成阻塞费用
- 阻塞费用用于补偿电网为解决网络阻塞而产生的成本
- 具体分摊方式由市场规则另行规定

## 第八章 辅助服务市场

### 8.1 辅助服务品种

内蒙古电力市场辅助服务包括以下品种：

| 品种 | 蒙西电网 | 蒙东电网 | 说明 |

| 备用 | 现货规则预留接口 | 备用市场 | 保障容量充裕 |
| 需求侧响应 | 已开展 | 逐步推进 | 负荷型虚拟电厂参与 |
| 容量备用 | 容量合约 | 容量交易 | 保障长期容量充裕 |

### 8.2 调频辅助服务（蒙东，已建立市场）

**基本信息**

- 《蒙东电力辅助服务（调频）市场实施细则（试行）》已于2025年10月17日发布
- 服务提供主体：电力调度机构直调AGC单元
- 新建AGC单元试运期结束后即纳入调频市场管理范围

**调频市场规则**

- AGC综合性能指标Kp值：反映调频单元响应速度、调节精度等综合性能
- Kp值上限：暂定为2.0
- Kp值下限：提供AGC服务期间Kp值小于0.5，不予补偿
- 市场初期以前70%的最小Kp值作为报价基准

**调频补偿公式**

```
调频里程补偿 = 单次调用功率 × 单次调用时长 × 里程单价（元/kWh）
调频性能补偿 = 调频里程补偿 × Kp值
容量补偿 = 储能额定功率 × 补偿标准 × 投运率
```

**调频报价上限**

- 暂定为**15元/MW**

**调频费用分摊**

- 调频辅助服务市场补偿费用由以下主体共同分摊：
  - 电力用户用电量
  - 电网侧储能下网电量
  - 未参与电能量市场交易的上网电量

**控制区调频需求**

- 应不小于负荷预测功率的**3%**
- 应不小于新能源预测功率的**20%**

### 8.3 调峰辅助服务（蒙东）

**调峰市场机制**

- 实时调峰交易采用"阶梯式"报价方式
- 火电机组按最大调峰能力进行分档申报
- 分档标准：额定容量100%-70%为一档，70%以下每10%为一档
- 报价方式：按价格递增方式逐档申报，每档全天报价相同

### 8.4 蒙西辅助服务现状

- 蒙西尚未建立独立的辅助服务市场
- 现货规则中已预留调频、备用等品种接口
- 独立储能可参与调频辅助服务市场（根据2026年3月相关文件）
- 调频市场独立运行后，储能充放电量按现货市场价格结算

### 8.5 需求侧响应交易

**基本信息**

- 2025年1月9日，内蒙古电力多边交易市场首次开展需求侧响应交易
- 早高峰（6:00-9:00）：中标电力100万千瓦，中标价格1.46元/千瓦时
- 晚高峰（17:00-24:00）：中标电力217万千瓦，中标价格1.50元/千瓦时
- 累计中标响应电量1901万千瓦时，实际响应电量2536万千瓦时
- 综合度电收益1.39元/千瓦时

**规则要求**

- 负荷型虚拟电厂通过聚合形式参与需求侧响应交易
- 储能资源与负荷资源捆绑参与
- 须按《内蒙古电力多边交易市场需求侧响应交易实施细则》执行

**日前需求侧响应**

- 申报响应价格下限：**100元/MWh**
- 申报响应价格上限：**1,500元/MWh**
- 响应容量最小单位：**1MW**
- 申报响应价格最小单位：**10元/MWh**
- 市场初期参与响应经营主体只能进行单一报价

## 第十章 市场风险与监管

### 10.1 市场风险类型

**价格风险**

- 现货市场价格波动：蒙西最大峰谷价差达1,549.86元/MWh
- 负电价风险：2025年11月已出现2天全天均价负电价
- 偏差结算风险：签约比例不足将产生20%偏差考核费用

**信用风险**

- 售电公司履约风险：须缴纳履约保函或履约保险
- 新型主体经营风险：独立储能、虚拟电厂商业模式尚在探索

**操作风险**

- 网络安全风险：电力交易平台面临攻击、干扰和破坏等行为
- 交易安全风险：电力交易机构可对疑似攻击主体访问平台进行限制

**市场力风险**

- 操纵市场价格：严格禁止，市场成员须自觉自律
- 损害其他市场成员合法权益：严格禁止

### 10.2 风险防控机制

**履约保障机制**

- 售电公司须向电力交易机构及时、足额缴纳履约保函或履约保险
- 新型主体须持续满足注册条件
- 电力交易机构加强零售市场运营管理
