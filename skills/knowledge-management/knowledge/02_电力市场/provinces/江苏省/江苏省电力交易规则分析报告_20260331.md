# 江苏省电力交易规则分析报告

## 第一章 市场主体概览

### 第一节 市场主体类型

| 主体类型 | 细分类型 | 准入条件 | 政策依据 |
|---------|---------|---------|---------|
| 发电企业 | 燃煤电厂、核电、燃气、风电/光伏 | 并网许可、环保达标、纳入调度 | 苏监能市场〔2026〕14号 |
| 售电公司 | 独立售电公司、电网企业售电 | 注册资本、履约保函、人员、系统 | 苏发改能源发〔2024〕1346号 |
| 电力用户 | 工商业用户、居民/农业用户 | 完成市场注册、计量装置合规 | 苏监能市场〔2026〕14号 |
| 新型主体 | 独立储能、虚拟电厂（负荷聚合商）、智能微电网 | 具备调节能力、完成注册 | 苏监能市场〔2026〕14号 |
| 电网企业 | 输配电、调度机构 | 国有企业，承担保底供电 | 苏监能市场〔2026〕14号 |

### 第二节 新型主体准入条件

**独立储能：**

| 参数 | 要求 | 政策依据 |
|------|------|---------|
| 放电功率 | ≥10 MW | 江苏调频辅助服务规则（征求意见稿，2025-10-31） |
| 持续放电时间 | ≥2 小时 | 同上 |
| 商业运营 | 必须进入商业运营 | 同上 |
| AGC能力 | 必须具备 | 同上 |
| 市场注册 | 在电力交易中心完成 | 苏发改能源发〔2024〕1346号 |

**虚拟电厂：**

| 参数 | 要求 | 政策依据 |
|------|------|---------|
| 聚合可调节能力 | ≥1 MW（参照周边省份） | 待江苏专项规则明确 |
| 响应时间 | ≤15分钟（参照周边省份） | 待江苏专项规则明确 |
| 注册 | 在电力交易中心完成 | 苏监能市场〔2026〕14号 |

### 第三节 售电公司准入条件

| 参数 | 要求 | 政策依据 |
|------|------|---------|
| 注册资本 | 2000万元以上（参照周边省份） | 苏发改能源发〔2024〕1346号 |
| 履约保函 | 按运营评价等级差异化缴纳 | 同上 |
| 从业人员 | 专职人员（须具备电力市场知识） | 同上 |
| 运营系统 | 具备技术支持系统 | 同上 |

### 第四节 注册与退出流程

| 环节 | 要求 | 时限 | 政策依据 |
|------|------|------|---------|
| 注册平台 | 江苏电力交易平台 | — | 苏监能市场〔2026〕14号 |
| 提交材料 | 营业执照、法人、资产证明、人员证明、运营系统等 | 即时 | 同上 |

| 生效 | 材料齐全后生效 | 即时 | 同上 |
| 自愿退出 | 提前申请，结清费用 | — | 同上 |
| 强制退出 | 违规、欠费、重大安全事故 | — | 同上 |

| 交易品种 | 交易方式 | 交易周期 | 价格机制 | 政策依据 |
|---------|---------|---------|---------|---------|
| 多年期交易 | 双边协商 | 多年（2-5年） | 协商定价 | 苏监能市场〔2026〕14号 |

| 参数 | 数值 | 说明 | 政策依据 |

## 第三章 现货市场规则

### 第一节 市场运行状态

| 参数 | 状态 | 政策依据 |

| 出清方式 | 报量报价，集中优化出清 | 江苏现货市场V2.1 |

| 市场 | 环节 | 时间 | 政策依据 |

| 日前 | 出清（SCUC+SCED） | D-1日 | 同上 |
| 日前 | 结果发布 | D-1日 | 同上 |
| 实时 | 调度执行（SCED） | D日实时 | 同上 |
| 实时 | 结算 | 月度 | 同上 |

| 参数 | 数值 | 说明 | 政策依据 |

| 报价下限 | **0元/兆瓦时** | — | 同上 |
| 出清价格 | 节点电价（初期全网统一） | — | 苏发改能源函〔2025〕407号 |

### 第四节 独立储能现货规则

| 参数 | 要求 | 政策依据 |
|------|------|---------|

| 充放电申报 | 须在日前市场申报充放电计划 | 江苏现货规则V2.1 |
| SOC管理 | 须申报荷电状态上下限 | 同上 |

> ⚠️ 参与现货后，不再适用775号文固定充放电价格。

## 第四章 零售市场规则

### 第一节 零售套餐分类体系

| 分类维度 | 类型 | 政策依据 |
|---------|------|---------|
| 价格形成机制 | 固定价格套餐、价格联动套餐、混合类套餐 | 江苏电力交易中心2024年12月通知 |
| 能源类型 | 常规电套餐、绿电套餐（证电分离） | 苏监能市场〔2026〕14号 |
| 交易周期 | 年度、季度、月度套餐 | 同上 |
| 用户类型 | 工商业用户、居民用户、农业用户 | 同上 |

### 第二节 固定价格套餐限制（江苏特色）

| 参数 | 数值 | 说明 | 政策依据 |
|------|------|------|---------|
| 固定价格浮动上限 | **不超过燃煤基准价20%** | 上下浮动均受限 | 江苏电力交易中心2024年12月通知 |
| 固定价格套餐比例上限 | **不超过50%** | 须配套价格联动套餐 | 同上 |

> 江苏是全国少数明确限制固定价格套餐比例的省份，核心目的是传导批发市场价格风险，防止批零倒挂。

### 第三节 偏差考核豁免

| 套餐类型 | 是否免考核 | 政策依据 |
|---------|-----------|---------|
| 价格联动类套餐 | **免考核** | 江苏规则 |
| 固定价格类套餐 | 考核偏差 | 同上 |
| 混合类套餐 | 按套餐条款 | 同上 |

### 第四节 套餐价格公式

| 套餐类型 | 价格公式 | 政策依据 |
|---------|---------|---------|
| 固定价格套餐 | 结算电价 = 套餐约定固定价格 | 江苏规则 |
| 价格联动套餐 | 结算电价 = 批发市场均价 + 浮动服务费 | 同上 |
| 绿电套餐 | 结算电价 = 批发市场均价 + 环境溢价 | 苏监能市场〔2026〕14号 |
