## 第一章 政策体系全貌

### 1.1 政策迭代时间线

浙江独立储能市场规则经历了三个主要阶段：

| 阶段 | 时间 | 标志性文件 | 核心内容 |
|------|------|----------|---------|
| **初建期** | 2024年3-4月 | "1+5"体系发布 | 浙江首次构建现货市场规则体系，储能首次纳入市场 |
| **完善期** | 2025年11-12月 | 5份3.0版细则+3项修订 | 调频3.0版确立，按小时选择机制明确，储能入市机制清晰 |
| **成熟期** | 2026年2月至今 | 4份3.1版细则（5月实施）+2026年运行方案 | 衔接全国统一市场"1+N"体系，绿电规则整合，监管强化 |

**三个文件的关键定位**：

| 文件 | 发布时间 | 核心定位 |
|------|---------|---------|
| 《浙江电力现货市场规则》 | 2024年4月 | 总纲，确立"1+5"体系框架，储能首次获得市场主体地位 |
| 《2026年浙江电力现货市场运行方案》 | 2025年11月27日 | 2026年全年执行，明确价格参数、二级限价、各主体参与方式 |
| 《浙江电力市场结算实施细则（3.1版）》 | 2026年2月24日（正式发布） | 2026年5月实施，细化储能充放电双重身份结算、绿电环境价值 |

### 1.2 2026年3.1版细则"一调整、三明确、五新增"

2026年2月24日，浙江电力市场管理委员会发布4份3.1版细则（2026年5月正式实施），是在3.0版基础上的精细化修订：

**一调整**
- 绿色电力交易专章整合进《浙江电力中长期市场实施细则》，不再单独成章，实现规则统一

**三明确**
1. **明确绿电合约分解详细流程**：发电户号→发电企业→售电公司→零售用户，形成完整合约链，满足欧盟等经济体绿电溯源合规要求
2. **明确售电公司履约保证递交时限和履约异常评分标准**：按月满足履约保证初始额度要求，未及时缴纳欠费将扣分直至信用等级降级
3. **明确绿电环境价值计算公式和结算周期**：绿电环境价值由环境价值费用+环境价值补偿费用组成，**次次月结算**

**五新增**
1. **合同签订验证机制**：未勾选封顶条款的零售合同，签订时需法定代表人人脸识别确认
2. **售电公司可交易电量限额规则**：零售交易限额=min{注册资产总额对应规模，履约担保额度对应规模}，签约时按规则扣减
3. **推送居民和农业用户月度电量**：进一步完善结算全数据体系
4. **"过户销户"独立章节**：明确各类主体过户销户时结算科目和时限
5. **中长期阻塞费用分配调节机制**：统调煤电中长期合约阻塞费用按月单独结算

### 1.3 独立储能的"三层市场身份"

| 身份层级 | 具体含义 | 适用规则 |
|---------|---------|---------|
| **电力用户** | 充电时段，储能作为电力用户从电网购电 | 支付充电电费，免输配电费（待确认） |
| **发电企业** | 放电时段，储能作为发电企业向电网售电 | 收取放电电费，按发电侧结算 |
| **新型经营主体** | 独立储能、虚拟电厂、负荷聚合商等列入新型主体 | 适用专项市场规则 |

> **注**：国家发改委规定，独立储能放电时段按发电企业身份参与交易，充电时段按电力用户身份参与交易。

### 2.1 准入条件

| 条件类型 | 具体要求 | 政策依据 |
|---------|---------|---------|
| **独立法人资格** | 具有独立法人资格，独立签订市场合同 | 市场管理实施细则3.1版 |
| **并网调度协议** | 与电网企业签订并网调度协议 | 市场管理实施细则3.1版 |
| **独立计量** | 具备电力、电量数据分时计量与传输条件，数据准确性满足结算要求 | 市场管理实施细则3.1版 |
| **调度自动化接入** | 接入电力调度自动化系统，统一监控和调度 | 市场管理实施细则3.1版 |
| **AGC调节能力** | 具备AGC调节能力，可接受调度实时控制 | 2026年运行方案 |
| **SOC实时反馈** | 实时反馈电池SOC状态、上下调频能力等信号至调度 | 调频细则3.0版 |
| **并网测试** | 完成并网测试，认定具备充放电能力 | 现货实施细则3.0版 |
| **涉网试验** | 完成涉网试验，认定具备自动调节和辅助服务能力 | 现货实施细则3.0版 |
| **注册手续** | 在浙江电力交易中心完成注册 | 市场管理实施细则3.1版 |
| **充放电技术条件** | 满足最大充放电功率、最大调节容量及持续充放电时间等技术条件 | 市场管理实施细则3.1版 |

### 2.2 配建储能转为独立储能

| 情形 | 规则 | 说明 |
|------|------|------|
| 配建储能选择 | 配建储能与所属经营主体视为一体 | 整体参与，不单独入市 |
| 转为独立储能 | 具备独立计量、控制等技术条件，具有法人资格时，可选择转为独立储能项目 | 作为经营主体直接参与市场 |

### 2.3 注册流程

```
步骤1：签订并网调度协议
步骤2：完成并网测试（认定充放电能力）
步骤3：完成涉网试验（认定辅助服务能力）
步骤4：在浙江电力交易中心注册
步骤5：签订入市协议
步骤6：按月维持履约保证初始额度（3.1版新增）
步骤7：参与市场（D-1日报申选择）
```

## 第三章 交易机制

### 3.1 核心规则：按小时动态选择机制

> ⚠️ **重要澄清**：浙江储能并非"二选一"（永久选择），而是**按小时动态选择**——每日D-1申报阶段，储能按小时申报次日是参与电能量市场还是调频市场，每日每个小时可以不同。

#### 机制详解

| 维度 | 规则 |
|------|------|
| **选择频率** | 每小时可选，每日重新申报 |
| **选择方式** | D-1日12:00前完成D日24小时各小时的选择意向申报 |
| **调频总容量上限** | 200MW（储能调频市场总容量上限） |
| **入选调频的时段** | 该小时不参与电能量市场 |
| **其余时段** | 按申报的自计划曲线参与电能量市场 |
| **SOC门槛** | 若小时内未参与电能量市场且满足SOC门槛要求，可参与该小时调频 |

#### D-1日完整流程

```
D-1日 9:30前  调度机构披露D日电网侧储能可调容量
D-1日 12:00前 储能完成D日各小时选择申报（调频 or 电能量？）
D-1日 16:00前 调度机构发布调频市场出清结果
              → 入选调频的储能（累计≤200MW）：该小时不参与电能量市场
              → 未入选或剩余储能：按自计划曲线参与电能量市场
D日 0-24时    执行申报结果
D+1日        日清分结算
```

#### 实际操作示例（50MW储能某典型日）

| 时段 | 时长 | 选择 | 策略理由 |
|------|------|------|---------|
| 0:00-7:00 | 7h | 电能量市场（充电） | 低谷电价，深夜充电 |
| 7:00-11:00 | 4h | 调频市场 | 早高峰调频需求高，收益更高 |
| 11:00-13:00 | 2h | 电能量市场（放电） | 高峰电价放电套利 |
| 13:00-17:00 | 4h | 调频市场 | 光伏午间低价时段，优先调频 |
| 17:00-21:00 | 4h | 电能量市场（放电） | 晚高峰高电价 |
| 21:00-24:00 | 3h | 调频市场 | 灵活调配 |

### 3.2 电能量市场交易规则

| 维度 | 规则 | 说明 |
|------|------|------|
| **参与方式** | **自计划方式**申报曲线 | 储能自主申报充放电计划，**作为价格接收者**参与市场 |
| **申报截止** | D-2日16:00前提交D日出力的日计划曲线 | 2026年运行方案 |
| **曲线要求** | 96点或其他规定曲线 | 每15分钟一点 |
| **日前定价** | 不参与市场定价（自计划方式） | 价格由火电/新能源等定价主体确定 |
| **实时市场** | 采用日前封存的申报信息 | 储能按日前申报曲线执行 |
| **一次调频联动** | 系统一次调频能力不足时，调度可设储能申报出力上限 | 保障系统安全 |
| **容量租赁穿透** | 扣除租赁部分容量，租赁部分不能用于市场申报 | 租赁合同需明确使用边界 |
| **常设申报** | 未按时申报，采用常设申报作为申报信息 | 防止迟报漏报 |

### 3.3 调频辅助服务市场规则（3.0版）

#### 出清机制

| 维度 | 规则 |
|------|------|
| **出清方式** | 按**调整排序价格**出清（综合考虑报价+机会成本） |
| **出清价格** | 入选者中最高调整后里程报价确定为出清价格（统一边际出清） |
| **火电调频里程报价上限** | **15元/MW** |
| **储能调频报价** | **无常设报价上限**（储能专属优势） |

#### 储能专项规则

| 维度 | 规则 |
|------|------|
| **调频总容量上限** | **200MW** |
| **申报容量下限** | 不得低于自身装机容量的**80%** |
| **申报容量上限** | 不得高于实际充放电最大功率 |
| **综合调频性能指标最大值** | **2** |
| **历史调频性能准入门槛** | **≥0.4**（低于此值无法参与） |
| **SOC要求** | 满足SOC门槛要求方可参与该小时调频 |
| **常设报价** | **无** |

#### 补偿计算公式

```
调频里程补偿 = 实际调频里程 × 出清价格 × 综合调频性能指标
```

**储能优势**：
- 响应速度快、调节精度高，性能指标可达2（火电约1）
- 无15元/MW报价上限，可更高报价竞争

### 3.4 中长期交易规则（3.1版）

| 维度 | 规则 |
|------|------|
| **绿电交易** | 专章整合入中长期细则（3.1版调整），不再单独成章 |
| **合约分解链** | 发电户号→发电企业→售电公司→零售用户（3.1版明确） |
| **环境价值** | 环境价值费用+环境价值补偿费用，**次次月结算**（3.1版明确） |
| **签约方式** | 双边协商、集中竞价、挂牌（售方只允许售出，购方只允许购入） |
| **滚动撮合** | 购售双方均可买入或卖出电量（3.1版明确） |
| **中长期占比** | **年度≥70%，月度累计≥90%**（2026年运行方案） |

### 3.5 2026年运行方案关键价格参数

| 参数类型 | 数值 | 说明 |
|---------|------|------|
| 现货**申报**价格下限 | **-200元/MWh** | 深度负电价，可充电获益 |
| 现货**申报**价格上限 || 现货**申报**价格上限 | **800元/MWh** | |
| 现货**出清**价格上限 | **1200元/MWh** | 极端行情触发二级限价 |
| 调频里程申报下限 | **0元/MW** | |
| 调频里程申报上限 | **15元/MW**（仅火电） | 储能无常设上限 |
| 中长期签约比例 | 年度≥70%，月度累计≥90% | 2026年运行方案 |

## 第四章 调度运行

### 4.1 调度管理要求

| 要求 | 说明 |
|------|------|
| **调度机构** | 国网浙江省电力公司调度中心（省调） |
| **调度指令** | 下达储能充放电计划和出力指令 |
| **AGC投运** | 现货市场运行期间，所有参与市场机组原则上须投入AGC远控模式 |
| **SOC实时反馈** | 须实时反馈电池SOC状态、上下调频能力等信号至调度 |
| **运行监控** | 接入调度自动化系统统一监控、调度 |
| **遵循指令** | 遵循调度指令执行充（放）电 |

### 4.2 并网运行考核

| 考核维度 | 内容 |
|---------|------|
| 计划曲线执行 | 日前申报曲线与实际执行偏差考核 |
| 可用率 | 储能可用时间占全运行时间比例 |
| 响应性能 | 调节速率、响应时间、调节精度 |
| 数据报送 | 运行数据实时报送 |
| 调度纪律 | 无故不执行调度指令按《华东区域发电厂并网运行管理实施细则》从重处罚 |

### 4.3 特殊情形处理

| 情形 | 处理方式 |
|------|---------|
| 安全风险（气候异常/电网故障） | 调度按安全第一原则处理，必要时中止现货市场，恢复非现货模式调度 |
| 市场操纵风险 | 能源监管办/发改委可中止现货市场 |
| 巨额盈亏风险 | 可中止运行或调整规则并重新出具结算依据 |
| 负备用不足 | 部分机组解除AGC，人工调度至最小技术出力以下，按市场价格下限出清 |

## 第五章 结算规则

### 5.1 结算模式："日清月结"

```
实时运行（15分钟/段）
  → D+1日：日清分，发布电能量和市场化辅助服务日清分结果
  → 月底：月度结算，形成月度结算依据
  → 次次月：绿电环境价值结算（3.1版新增）
  → 生成月电费结算账单，进行电费收付
```

### 5.2 储能双重身份结算体系

#### 放电时段（作为发电企业）

| 结算项目 | 计算方式 |
|---------|---------|
| 放电电能量电费 | 放电日前电费 + 放电实时偏差电费 + 放电中长期差价电费 |
| 市场化辅助服务收入 | 调频里程补偿（如参与调频） |
| 成本补偿 | 启动成本、空载成本、电能成本（当收入无法覆盖报价成本时） |

#### 充电时段（作为电力用户）

| 结算项目 | 计算方式 |
|---------|---------|
| 充电电能量电费 | 充电日前电费 + 充电实时偏差电费 + 充电中长期差价电费 |
| 市场分摊费用 | 按市场化辅助服务费用分摊、成本补偿费用分摊等 |
| 输配电费 | 充电免输配电费（待官方确认） |

### 5.3 ⭐ 二级限价机制（2026年运行方案·核心风险点）

| 维度 | 规则 |
|------|------|
| **监测值定义** | 现货市场统调发电侧月度平均电能量价格（含日前+实时电能量电费+中长期差价+曲线偏差损益+水电偏差回收等） |
| **触发条件** | 监测值**高于**二级限价触发值（触发值按季度政府通知） |
| **日度触发** | 日前+实时市场统一结算点出清价格算术平均值超标时，调整当日出清价格 |
| **月度触发** | 月度监测值超标时，同比例调整**全月**日前和实时出清价格，直至监测值≤触发值 |
| **调整范围** | 电能量电费、中长期差价、曲线偏差损益、运行成本补偿、超额获利回收、零售套餐参考价等 |
| **披露价格** | 信息披露价不变，仅结算价调整 |
| **触发值来源** | 根据统调燃煤电厂电煤到广均价确定，按季度通知 |

> ⚠️ **对储能的影响**：煤价大涨触发二级限价时，储能全月结算价格被系统性压低，放电收益和调频收益均受损。这是2026年运行方案中储能最重要的风险点。

### 5.4 储能市场分摊/返还费用

| 费用类型 | 规则 |
|---------|------|

## 第八章 风险提示与实操要点

### 8.1 主要风险

| 序号 | 风险类型 | 详情 | 应对建议 |
|------|---------|------|---------|
| 1 | **⭐ 二级限价触发** | 煤价大涨→月度均价超标→全月结算价被调低 | 关注煤价走势，预留收益安全边际；关注季度触发值通知 |
| 2 | **调频容量上限200MW** | 储能调频总容量有上限，竞争激烈时可能无法全时段入选 | 提升综合调频性能（目标≥1.5）；优化申报策略 |
| 3 | **综合性能不达标** | 低于0.4无法参与调频市场 | 优化BMS/EMS策略，确保调节性能达标 |
| 4 | **SOC管理** | 参与调频须满足SOC门槛，SOC过高或过低均无法调频 | 制定精细化SOC管理曲线，平衡充放电与调频 |
| 5 | **履约保证资金压力** | 3.1版要求按月核查履约保证初始额度 | 提前规划资金，确保按时缴纳 |
| 6 | **年利用小时不达标** | 容量补偿要求≥600小时/年，不达标将影响后续年度补偿 | 加强运维，确保年利用小时达标 |
| 7 | **深度负电价** | 申报下限-200元/MWh，低谷时段可能出现负电价 | 充电策略需考虑负电价时段的收益机会 |
| 8 | **常设申报风险** | 迟报漏报默认常设报价，可能导致不利价格 | 建立D-1日申报提醒机制 |

### 8.2 3.1版过渡期实操建议（2026年5月前）

| 序号 | 事项 | 具体动作 |
|------|------|---------|
| 1 | **履约保证核查** | 对照3.1版要求，核查履约保证初始额度是否满足按月要求 |
| 2 | **D-1日9:30信息披露** | 建立机制，每日9:30查看调度披露的储能可调容量，优化次日申报策略 |
| 3 | **SOC精细化管理** | 提升BMS/EMS系统性能，确保SOC实时准确，满足调频门槛要求 |
| 4 | **绿电合约链梳理** | 若涉及绿电，建立完整合约链（发电户号→发电企业→售电公司→零售用户） |
| 5 | **资金安排** | 提前安排履约保证资金，避免因资金问题影响市场参与资格 |
| 6 | **收益安全边际** | 在收益测算时预留10%-15%安全边际，防范二级限价触发风险 |
