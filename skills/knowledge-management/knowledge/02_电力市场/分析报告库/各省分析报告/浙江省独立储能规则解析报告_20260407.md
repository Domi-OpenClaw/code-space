## 一、新型主体基本情况

### 1.1 定义与核心特征

**定义**：浙江省独立新型储能电站是指具备独立法人资格、直接接入电网运行、以独立市场主体身份参与电力市场的电化学储能设施。不依附于发电企业或用户侧项目，单独履行注册入市手续，独立参与电能量市场和辅助服务市场。

**政策依据**：
- 《浙江电力现货市场规则》（2024年4月正式印发）
- 《浙江电力现货电能量市场交易实施细则（3.0版）》（2025年11月）
- 《浙江电力调频辅助服务市场交易实施细则（3.0版）》（2025年11月28日）
- 《浙江电力市场管理实施细则（3.0版）》（2025年12月修订）
- 《浙江省新型储能项目管理办法（暂行）》（征求意见稿，2025年11月）

**核心特征**：

| 序号 | 特征 | 说明 | 政策依据 |
|------|------|------|---------|
| 1 | 独立法人资格 | 必须具有独立法人资格，独立签订市场合同 | 市场管理实施细则 |
| 2 | 双重市场身份 | 放电时段为发电企业，充电时段为电力用户 | 国家发改委专项规定 |
| 3 | 容量租赁穿透 | 扣除租赁部分容量后方可参与市场 | 现货实施细则 |
| 4 | **"二选一"机制** | 储能须在调频市场与电能量市场中选择其一参与 | 调频实施细则3.0版 |
| 5 | 并网测试门槛 | 完成并网测试认定充放电能力，完成涉网试验认定辅助服务能力 | 现货实施细则 |
| 6 | 自计划申报 | 以自计划方式申报曲线 | 现货实施细则 |

## 二、市场准入规则

### 2.1 准入条件

| 条件类型 | 具体要求 | 政策依据 |
|---------|---------|---------|
| **额定功率** | 参照浙江新型储能项目管理办法（电网侧储能需满足准入条件） | 《浙江省新型储能项目管理办法（暂行）》征求意见稿 |
| **持续充放电时间** | 额定功率低于5MW的用户侧储能不纳入规模管控 | 同上 |
| **并网测试** | 完成并网测试且检测结果合格，认定具备充放电能力 | 现货实施细则 |
| **涉网试验** | 完成涉网试验且试验结果合格，认定具备自动调节和辅助服务能力 | 现货实施细则 |
| **调度控制** | 可接受调度部门运行控制 | 现货实施细则 |
| **容量要求** | 扣除租赁部分容量后方可参与市场（若涉及租赁） | 现货实施细则 |
| **注册手续** | 在浙江电力交易中心完成注册 | 市场管理实施细则 |

> ⚠️ **注意**：浙江对独立储能的功率门槛未在公开文件中明确为"≥5MW"，而是采用"5MW以下用户侧储能不纳入规模管控"的表述，实际准入功率要求需以浙江电力交易中心最新注册要求为准。

### 2.2 电网侧储能特殊规定

| 维度 | 规则 | 说明 |
|------|------|------|
| 准入条件 | 满足规定准入条件，并在交易中心完成注册手续 | 可参与市场化交易 |
| 规模管控 | 额定功率低于5MW的用户侧新型储能不纳入规模管控 | 用户侧轻量发展 |
| 电源侧储能 | 应与发电项目为同一整体，参与市场化交易无需单独注册 | 打包处理 |
| 用户侧储能 | 应与用电项目为同一整体，参与市场化交易无需单独注册 | 打包处理 |

### 2.3 注册流程

```
并网测试（认定充放电能力）→ 涉网试验（认定辅助服务能力）→ 浙江电力交易中心注册 → 签订入市协议 → 参与市场
```

| 步骤 | 内容 | 说明 |
|------|------|------|
| 1 | 并网测试 | 认定具备充放电能力，可接受调度控制 |
| 2 | 涉网试验 | 认定具备自动调节和提供辅助服务能力 |
| 3 | 交易中心注册 | 提交相关材料，获取市场主体资格 |
| 4 | 签订入市协议 | 明确权利义务 |
| 5 | 正式参与市场 | 可选择参与电能量市场或调频市场（二选一） |

## 三、交易规则

### 3.1 交易品种

| 交易品种 | 参与方式 | 申报规则 | 政策依据 |
|---------|---------|---------|---------|
| **电能量市场（现货）** | 自计划方式申报曲线 | 申报充放电曲线 | 现货实施细则3.0版 |
| **调频辅助服务市场** | 报量报价，里程出清 | 申报调频容量+里程报价 | 调频实施细则3.0版 |
| **中长期交易** | 分时段签约 | 分时段交易 | 中长期交易实施细则3.0版 |
| **容量补偿** | 自动获得（电网侧项目） | 核定容量×200元/kW·年 | 容量补偿分配方案 |

> ⭐ **浙江核心特色：储能"二选一"机制**
> 储能电站在**电能量市场**和**调频辅助服务市场**中**须选择其一参与，不得同时参与两者**。这是浙江独有的规则设计，旨在优化储能资源分配。

### 3.2 ⭐ 调频辅助服务市场规则（3.0版·核心亮点）

2025年11月28日，浙江电力交易中心发布《浙江电力调频辅助服务市场交易实施细则(3.0版)》，这是全国最成熟的省级调频市场规则之一：

#### 出清机制

| 维度 | 规则 | 说明 |
|------|------|------|
| 出清方式 | **按调整排序价格出清** | 综合考虑报价+机会成本 |
| 出清价格确定 | 入选者中**最高调整后里程报价**确定为出清价格 | 统一边际出清 |
| 里程报价上限 | **火电：15元/MW** | 有明确上限 |
| 储能报价 | **无常设报价上限** | 相比火电有更大报价空间 |

#### 储能专项规则

| 维度 | 规则 |
|------|------|
| **调频容量上限** | **200MW** |
| **申报容量下限** | 不得低于自身装机容量的**80%** |
| **申报容量上限** | 不得高于实际充放电最大功率 |
| **综合调频性能指标最大值** | **2** |
| **历史调频性能指标准入门槛** | **0.4** |
| 常设报价 | **无**（储能专属优势） |

#### 补偿计算

```
调频里程补偿 = 实际调频里程 × 出清价格 × 综合调频性能指标
```

> 💡 **储能调频优势解读**：
> - 火电受15元/MW上限约束，储能无此限制，可以更高报价竞争
> - 储能响应速度快、调节精度高，综合调频性能指标可达2（火电一般1左右）
> - 但需确保申报容量的80%-100%在范围内

### 3.3 电能量市场规则

| 维度 | 规则 | 说明 |
|------|------|------|
| 参与方式 | **自计划方式申报曲线** | 储能自主申报充放电计划 |
| 曲线申报 | 按调度要求申报96点或其他规定曲线 | 日前市场申报 |
| 一次调频联动 | 当系统一次调频能力不足时，调度可设储能申报出力上限 | 保障系统安全 |
| 容量租赁穿透 | 扣除租赁部分容量 | 租赁部分容量不能用于自计划申报 |

### 3.4 中长期交易

| 维度 | 规则 |
|------|------|
| 签约方式 | 分时段交易（峰谷平段分别签约） |
| 结算方式 | 分时段计量、分时段结算 |
| 偏差处理 | 偏差电量按现货价格结算 |

## 四、调度与运营

### 4.1 调度管理

| 调度维度 | 规则 |
|------|------|
| 调度机构 | 省调（国网浙江省电力公司调度中心） |
| 调度指令 | 下达储能充放电计划和出力指令 |
| 出力上限干预 | 一次调频能力不足时可设储能申报出力上限 |
| 调度控制 | 储能须可接受调度部门运行控制 |

### 4.2 并网运行考核

参照浙江电力并网运行管理实施细则，主要考核维度：

| 考核维度 | 内容 |
|---------|------|
| 计划曲线执行 | 日前申报曲线与实际执行偏差考核 |
| 可用率 | 储能可用时间占全运行时间比例 |
| 响应性能 | 调节速率、响应时间、调节精度 |
| 数据报送 | 运行数据实时报送要求 |

### 4.3 运营要求

| 要求 | 说明 |
|------|------|
| **并网测试** | 认定具备充放电能力，可接受调度控制 |
| **涉网试验** | 认定具备自动调节和提供辅助服务能力 |
| **运行控制** | 须可接受调度部门运行控制 |
| **数据报送** | 实时运行数据报送至调度和交易中心 |
| **容量管理** | 租赁部分容量需从市场中扣除 |

## 五、结算规则

### 5.1 结算流程

```
实时运行 → 日清分 → 月度结算 → 费用支付
```

| 步骤 | 内容 | 周期 |
|------|------|------|
| 1 | 实时运行数据记录 | 实时/15分钟 |
| 2 | 日清分 | D+1日 |
| 3 | 月度结算确认 | 月底 |
| 4 | 费用支付 | 按月支付 |

### 5.2 结算价格

| 结算品种 | 结算价格 | 说明 |
|---------|---------|------|
| **现货电能量** | 节点电价 | 实时市场出清价格 |
| **中长期合约** | 合约约定价格 | 分时段结算 |
| **调频里程** | 出清价格×综合性能指标 | 按里程×性能系数 |
| **容量补偿** | **200元/kW·年** | 电网侧储能核定容量 |

### 5.3 双重身份结算

| 时段 | 身份 | 结算方式 |
|------|------|---------|
| 充电时段 | 电力用户 | 按用户侧价格结算（免收输配电费待确认） |
| 放电时段 | 发电企业 | 按发电侧节点电价结算 |

## 七、实操要点

### 7.1 实操案例

**案例 1：浙江衢州80MW/160MWh独立储能电站（2026年3月并网）**

- **背景**：浙江首座电网侧独立大型共享储能电站，总规模80MW/160MWh
- **操作步骤**：
  1. 完成并网测试（认定充放电能力）
  2. 完成涉网试验（认定辅助服务能力）
  3. 在浙江电力交易中心注册

### 7.3 常见问题（FAQ）

**Q1：浙江独立储能的"二选一"机制具体是什么？**
A：储能电站在参与市场时，必须选择在**调频辅助服务市场**或**电能量现货市场**中的一个作为主要参与形式，**不得同时参与两者**。选择调频市场则专注提供调频服务获得里程补偿；选择电能量市场则通过峰谷价差套利获得收益。

**Q2：浙江储能调频相比火电有哪些优势？**
A：主要有三点：①火电调频里程报价上限15元/MW，储能无常设上限，可以更高报价；②储能响应速度和调节精度更高，综合调频性能指标可达2（高于火电的1左右）；③50MW储能调频容量上限200MW，足够覆盖大多数项目。

**Q3：浙江容量补偿200元/kW·年如何计算？**
A：以50MW储能电站为例：50,000kW × 200元/kW·年 = 1,000万元/年。浙江17个项目首年共获2.328亿元补偿，平均单个项目约1,370万元。

**Q4：浙江储能参与市场需要完成哪些测试？**
A：两阶段测试：①并网测试（认定具备充放电能力，可接受调度控制）；②涉网试验（认定具备自动调节和提供辅助服务能力）。两个测试都通过后才能正式参与市场。

**Q5：浙江5MW以下的储能项目有什么特殊政策？**
A：额定功率低于5MW的用户侧新型储能不纳入规模管控，但需要与用电项目为同一整体参与市场化交易，无需单独注册。

**Q6：浙江独立储能的收益路径有哪些？**
A：主要三条：①容量补偿（200元/kW·年，稳定收益）；②调频市场（性能好则收益高）；③现货套利（峰谷价差收益）。建议根据电站特性选择最优路径。
