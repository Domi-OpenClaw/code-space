# new-power-subjects-rule Skill 概要

## 核心定位
新型主体电力市场准入规则 BCF 层业务专家 Skill，支撑虚拟电厂/储能等新型主体规则分析报告。

## 版本
V2.2.0

## 覆盖主体类型
1. **独立储能** — 10MW/20MWh 起，充放电双向计量
2. **虚拟电厂** — 各省分类不同（山东4类/河南3类），禁止跨省套用
3. **微电网** — 源网荷储一体化
4. **充电设施** — 充电免输配电费

## 核心分析方法
**7 大类规则红利对比框架**（新型主体 vs 售电公司）：
- 中长期持仓自由度
- 偏差考核豁免（价格联动免考核）
- 节点电价优势
- 现货市场参与比例
- 辅助服务参与资格
- 容量补偿（0.096元/kWh）
- 输配电费减免
