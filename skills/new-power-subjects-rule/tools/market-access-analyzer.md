# Tool: market-access-analyzer

**职责**：解析市场准入规则（准入条件、注册流程、材料清单）

**版本**：V2.0.0（2026-03-22 优化）

**输出标准**：≥1000 字，≥3 张表，≥5 个政策依据

---

## 输入

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | string | ✅ | 省份名称 |
| `subject_type` | string | ✅ | 新型主体类型 |
| `policy_files` | array | ❌ | 政策文件列表 |

---

## 输出（第二章：市场准入规则）

### 2.1 准入条件（量化标准·≥1 张表·≥5 行）

**搜索关键词建议**：
- `{省份} {主体类型} 准入条件 门槛`
- `{省份} 虚拟电厂 聚合容量 注册资本`
- `{省份} 独立储能 功率 持续时间`


**注**：如该省有特殊要求（如公示期），需单独说明。

### 2.3 材料清单（≥1 个清单·≥5 项）

**材料清单模板**：
1. **基本注册资料**：
   - 注册申请表（加盖公章）
   - 营业执照复印件
   - 法人身份证复印件

2. **技术资料**：
   - 具备信息处理、运行监控、业务管理、计量监控、控制执行等功能的软硬件系统材料
   - 与电力调度机构签订的并网调度协议
   - 调节能力测试认定材料

3. **特殊材料**（如适用）：
   - 拥有配电网运营权的：安全生产和组织制度、电力业务许可证（供电类）等
   - 售电公司申请注册：基本注册资料无须重复提交

## 处理逻辑（5 步）

1. **搜索确认**：搜索该省准入条件政策文件（≥2 个独立来源）
2. **提取量化标准**：聚合容量、注册资本、人员资质等
3. **生成流程图和流程表**：注册流程详解
4. **整理材料清单**：分类列出（基本/技术/特殊）
5. **补充地方案例**：实操案例和地市差异

## 常见错误（避免）

| 错误 | 正确做法 |
|------|---------|
| 聚合容量门槛写"按规定执行" | 必须写具体数值（≥5MW） |
| 注册流程缺少办理时限 | 每个步骤必须标注时限 |
| 材料清单笼统 | 分类列出（基本/技术/特殊） |
| 无实操案例 | 至少提供 1 个实操案例 |

## 调用示例（山东独立储能）

**输入**：
- 省份：山东省
- 主体类型：独立储能
- 政策文件：[《山东电力市场规则（试行）》（征求意见稿，2026 年 3 月）]

**输出**：
```markdown
## 第二章 市场准入规则

### 2.1 准入条件

| 条件类型 | 具体要求 | 单位 | 政策依据 |
|---------|---------|------|---------|
| 聚合容量门槛 | 充放电功率≥5MW，持续充放时间≥2 小时 | MW/小时 | 《山东电力市场规则（试行）》 |
| 技术要求 | 具备信息处理、运行监控、业务管理、计量监控、控制执行等功能 | - | 《山东电力市场规则（试行）》 |
| 调度要求 | 与电力调度机构签订并网调度协议，具备调度直控条件 | - | 《山东电力市场规则（试行）》 |
| 安全要求 | 通过安全评估，具备可观、可测、可调、可控能力 | - | 《山东电力市场规则（试行）》 |
| 计量要求 | 依据电能量计量装置进行结算 | - | 《山东电力市场规则（试行）》 |

**注**：分布式储能准入门槛较低（≥1MW，持续≥2 小时），可独立参与或以聚合方式参与。

### 2.2 注册流程

```
申请注册 → 完整性审查（5 工作日）→ 签订入市协议 → 注册生效 → 推送清单
```

**注**：独立储能无需进行注册公示，与电力交易机构签订入市协议后注册生效。

| 步骤 | 办理部门 | 办理时限 | 所需材料 | 政策依据 |
|------|---------|---------|---------|---------|
| 申请注册 | 山东电力交易平台 | 即时 | 注册申请表、营业执照等 | 《山东电力市场规则（试行）》 |
| 完整性审查 | 电力交易机构 | 5 工作日 | - | 《山东电力市场规则（试行）》 |
| 签订入市协议 | 电力交易机构 | 即时 | 入市协议文本 | 《山东电力市场规则（试行）》 |
| 注册生效 | 电力交易机构 | 即时 | - | 《山东电力市场规则（试行）》 |
| 推送清单 | 电力交易机构 | 即时 | - | 推送至电网企业与电力调度机构 |

### 2.3 材料清单

1. **基本注册资料**：
   - 注册申请表（加盖公章）
   - 营业执照复印件
   - 法人身份证复印件

2. **技术资料**：
   - 具备信息处理、运行监控、业务管理、计量监控、控制执行等功能的软硬件系统材料
   - 与电力调度机构签订的并网调度协议
   - 调节能力测试认定材料

3. **拥有配电网运营权的储能**（如适用）：
   - 安全生产和组织制度
   - 电力业务许可证（供电类）
   - 配电区域网络分布证明材料
   - 配售电业务服务章程

4. **其他**：
   - 若售电公司申请注册，基本注册资料无须重复提交

### 2.4 地方差异及实操案例

**山东省特点**：
- **无需公示**：独立储能无需进行市场公示（虚拟电厂需公示 30 天）
- **快速注册**：完整性审查 5 工作日完成，签订入市协议即生效
- **单独注册**：分布式储能可单独进行市场注册

**实操案例**：
- **案例**：山东某 50MW/100MWh 独立储能电站
  - **背景**：2024 年 7 月投运，参与现货市场 + 调频辅助服务
  - **注册流程**：提交申请→5 工作日审查→签订入市协议→注册生效
  - **时间**：约 1-2 周完成全部注册流程
  - **关键**：提前准备技术资料，确保与调度机构签订并网调度协议
```
