# Tool: subject-basics-analyzer

**职责**：解析新型主体基本情况（定义、分类、特征、与售电公司差异）

**版本**：V2.0.0（2026-03-22 优化）

**输出标准**：≥800 字，≥3 张表，≥3 个政策依据

---

## 输入

| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `province` | string | ✅ | 省份名称（如"山东省"） |
| `subject_type` | string | ✅ | 新型主体类型（虚拟电厂/独立储能/智能微电网/充电设施运营商） |
| `policy_files` | array | ❌ | 政策文件列表 |

---

## 处理逻辑（5 步）

1. **搜索确认**：搜索该省分类方式政策文件（≥2 个独立来源）
2. **提取定义**：提取官方定义和核心特征
3. **生成对比表**：与传统主体对比（≥5 行×4 列）
4. **生成分类表**：该省实际分类方式（必须有文号）
5. **生成差异表**：与售电公司差异（≥6 行）

## 常见错误（避免）

| 错误 | 正确做法 |
|------|---------|
| 把山东 4 类套用到河南 | 每省单独查询，严禁套用 |
| 分类方式无文号支撑 | 必须标注政策文号，查不到写"待核实" |
| 聚合容量门槛写"按规定执行" | 必须写具体数值（≥X MW） |
| 与售电公司差异表<5 行 | 至少 6 行，覆盖核心差异 |

## 调用示例（山东独立储能）

**输入**：
- 省份：山东省
- 主体类型：独立储能
- 政策文件：[《山东电力市场规则（试行）》（征求意见稿，2026 年 3 月）]

**输出**：
```markdown
## 第一章 新型主体基本情况

### 1.1 定义与核心特征

**定义**：独立新型储能是指独立于新能源项目，以独立主体身份参与电力市场交易的储能设施（《山东电力市场规则（试行）》）。

**核心特征**：
1. **独立性**：独立于新能源项目，单独注册、单独计量、单独结算
2. **市场化**：以报量报价方式参与现货市场，自愿参与日前市场出清
3. **灵活性**：可选择同时参与电能量 + 辅助服务，或按日仅参与其中一种
4. **可控性**：具备调度直控条件，可实现可观、可测、可调、可控

### 1.2 分类方式

山东省储能分为 2 大子类：

| 子类类型 | 定义 | 准入条件 | 聚合容量门槛 | 政策依据 |
|---------|------|---------|-------------|---------|
| 独立新型储能 | 独立于新能源项目的储能设施 | 单独注册、单独计量 | 充放电功率≥5MW，持续充放时间≥2 小时 | 《山东电力市场规则（试行）》 |
| 分布式储能 | 分布式配置的储能设施 | 可独立或以聚合方式参与 | 充放电功率≥1MW，持续充放时间≥2 小时 | 《山东电力市场规则（试行）》 |

### 1.3 与售电公司的核心差异

| 对比维度 | 独立储能 | 售电公司 | 差异说明 | 规则红利 |
|---------|---------|---------|---------|---------|
| 市场定位 | 发用电一体化主体 | 电力购销中间商 | 储能可充电（用电）也可放电（发电） | - |
| 参与方式 | 报量报价参与现货市场 | 代理用户参与中长期市场 | 储能直接参与现货，售电公司主要参与中长期 | - |
| 收益来源 | 容量租赁 + 容量补偿 + 现货套利 + 辅助服务 | 购售电价差 | 储能收益来源多元化 | 收益更稳定 |
| 考核规则 | K 值考核（日可用容量核定） | 偏差考核 | 储能考核运行可靠性，售电公司考核电量偏差 | - |
| 结算价格 | 节点电价结算 | 统一结算点电价 | 储能按物理节点电价结算，更精准反映位置价值 | 可选择有利节点投资 |
| 中长期签约 | 无强制要求 | 年度≥80%、月度≥90% | 储能可自由选择中长期持仓比例 | 保留现货波动收益空间 |
```
